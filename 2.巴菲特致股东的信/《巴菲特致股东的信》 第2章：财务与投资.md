## A.市场先生

### 一、 核心投资原则

我们的投资方法适用于流通股与未上市公司，始终聚焦于三个维度：

| 评估维度         | 具体内涵                                   |
| :--------------- | :----------------------------------------- |
| **经济前景**     | 公司能否以令人满意的速度持续提升内在价值。 |
| **负责管理的人** | 管理层的能力与诚信。                       |
| **支付的价格**   | 买入价格是否具备安全边际。                 |

> 我们扮演的是**企业分析师**，而非市场、宏观经济或证券分析师。



### 二、 市场先生寓言

本杰明·格雷厄姆提出的“市场先生”概念，是隔离市场情绪的关键。他将市场想象成一位每日提供报价的合伙人：

| 市场先生的状态 | 行为表现                     | 报出价格             |
| :------------- | :--------------------------- | :------------------- |
| **情绪愉快**   | 只看到有利因素，怕你抢走利润 | **高价**（买卖均高） |
| **情绪低落**   | 只看到负面因素，怕你甩卖股权 | **低价**（买卖均低） |

💡 **核心启示**：
- 市场先生**服务于你**，而非指导你。他的钱袋比智慧更有用。
- 你可以无视他的报价，或利用其愚蠢的情绪。若受其影响，将是一场灾难。
- ⚠️ **关键前提**：若无法比市场先生更准确地评估公司价值，就不应参与这场游戏。



### 三、 投资成功所需的两种能力

成功的投资者必须同时具备：

1.  **判断优秀企业的能力** —— 识别能够持续提升内在价值的公司。
2.  **隔离情绪的能力** —— 将自身思维和行为与市场极易传染的情绪隔离开。

格雷厄姆的名言总结了市场规律：
> **短期来看，市场是一台投票机；长期来看，市场是一台称重机。**

企业价值的确认可能滞后，但这并非坏事，反而可能提供以更低价格增持的机会。



### 四、 永久持有与卖出准则

#### 4.1 永久持有的三个理由

我们将部分股票视为伯克希尔永久的组成部分，只要满足：

1.  **资产回报前景**令人满意。
2.  **管理层**能力优秀且为人诚实。
3.  **市场**没有出现高估。

#### 4.2 卖出场景

我们卖出持股，**绝非**因为股价上涨或持有时间过长。⚠️ “只要能赚钱，就不会破产”是华尔街最愚蠢的格言。卖出的真实原因包括：

-   市场**高估**公司价值。
-   需要资金抓住**更具吸引力的低估机会**或投资于**更为了解**的公司。
-   在极端必需的情况下，用于弥补保险业务的异常损失（极少发生）。



### 五、 净买家的正确心态

#### 5.1 汉堡包与汽车思维实验

| 你的角色   | 日常需求             | 你希望价格如何？ |
| :--------- | :------------------- | :--------------- |
| 终身消费者 | 汉堡包（不生产牛肉） | **更低**         |
| 定期消费者 | 汽车（不生产汽车）   | **更低**         |

#### 5.2 投资者的最终测验

若未来五年，你预期成为一名**净储蓄者（净买家）**，你应当希望股市**下跌**，而非上涨。原因在于：

-   **留存利润再投资**：伯克希尔通过留存收益代表股东进行“储蓄”和再投资，买入成本越低，该间接储蓄计划越有利。
-   **公司回购放大收益**：当持有的优秀公司进行回购时，股价越低，同等资金可回购的股份越多，我们在这些公司中的所有权比例提升就越显著。例如对可口可乐、富国银行在低价时的回购，使伯克希尔获益远超高价时的回购。

#### 5.3 重新解读新闻头条

| 常见认知（错误）         | 正确解读                                                     |
| :----------------------- | :----------------------------------------------------------- |
| “投资者因股市大跌而亏损” | “**撤资者**因股市大跌而亏损，但**投资者（净买家）在获益。**” |

> 每个买家对应一个卖家，一方的损失往往成就另一方的机会。



## B.套利

### 一、套利在投资组合中的定位

我们集团旗下的保险子公司有时会参与套利活动，作为手中**短期现金等价物的替代方式**，既保持高流动性，又能获得一定收益。我们的偏好是长期投资，但经常会遇到“现金多过好主意”的情况。此时，套利能提供比国债高得多的回报，更重要的是，它可以让我们面对放松长期投资标准的诱惑时冷静下来。

💡 **芒格语录**：“每当我们谈论完套利后，芒格常会总结道：‘OK，至少，它让你离开了酒吧。’”

**1988年套利业绩**：平均投资金额 **1.47亿美元**，税前获利 **0.78亿美元**，无论绝对额还是回报率都相当可观。



### 二、套利的定义与演变

| 时期           | 定义                                                         | 特征                                               |
| -------------- | ------------------------------------------------------------ | -------------------------------------------------- |
| 早期经典       | 在两个不同市场同时买卖证券或外汇，抓取微小差价               | 无风险或极低风险，如荷兰皇家石油在不同交易所的价差 |
| 现代“风险套利” | 利用已宣布的公司事件（出售、合并、重组、改制、清算、自我收购等）追逐利润 | 主要风险是事件未发生，与股市整体走向无关           |

⚠️ **注意**：自第一次世界大战后，套利含义已大幅扩展，如今所谓的“套利”多为风险套利。



### 三、经典早期案例：洛克伍德公司（1954年）

#### 背景

- 公司：洛克伍德公司（Rockwood & Co.），位于纽约布鲁克林，巧克力生产商
- 盈利有限，1941年起库存采用 **LIFO（后进先出）** 会计方法
- 1954年可可短缺，价格从 **5美分/磅** 飙升至 **超过60美分/磅**
- 若直接出售可可，需缴纳近 **50%** 的收益（账面留下的，都是成本**5美分/磅** ）

#### 交易结构

得益于1954年税务法中的特殊条款：如果将库存分配给股东，作为终止业务计划的一部分，LIFO 导致的利润可**免税**。

- 公司宣布终止可可奶油业务，**1300万磅** 可可豆供股东分配
- 同时以 **80磅可可豆换1股** 的比例回购公司股票

#### 操作与结果

巴菲特在格雷厄姆-纽曼公司工作时，年仅24岁的他：

- 持续买入股票，将股票证书换成仓库单据，然后卖出可可豆
- 利润非常可观，成本仅是几张地铁车票

该交易的设计师是当时默默无闻、年仅**32岁**的杰伊·普利兹克（Jay Pritzker）。洛克伍德股价从 **15美元飙升至100美元**，尽管公司存在巨大经营亏损。

💡 **启示**：在股票估值中，有比市盈率（PE）更有价值的东西。



### 四、套利评估框架：必须回答的四个问题

| 序号 | 核心问题                             | 说明                                     |
| :--: | ------------------------------------ | ---------------------------------------- |
|  1   | 承诺的事情发生的**概率**有多大？     | 评估交易完成的确定性                     |
|  2   | 资金会被**占用多久**？               | 影响年化回报率的关键                     |
|  3   | 出现**更好报价**（如竞购）的可能性？ | 潜在上行空间                             |
|  4   | 若预期事件**未发生**，后果如何？     | 包括反垄断、财务差错、融资失败等下行风险 |

任何套利决策都应围绕这四个问题展开严谨分析。



### 五、复杂案例全记录：阿克塔公司（Arcata Corp.）

> 💡 **关键词**：套利的四个问题及一个真实案例

#### 5.1 背景梳理

| 项目     | 详情                                                         |
| -------- | ------------------------------------------------------------ |
| 被收购方 | 阿克塔公司，主营印刷和森林木材                               |
| 收购方   | KKR（大型杠杆收购企业）                                      |
| 初始报价 | **37.50美元/股**                                             |
| 报价条件 | 获得“满意资金” + 红杉林索赔额外所得的 **2/3** 归属KKR        |
| 特殊资产 | 1978年美国政府征用 **17,000英亩** 红杉林，已付0.979亿美元，但利率、复利计算等存在争议 |

#### 5.2 套利四问分析

1. **概率**：KKR 过往记录良好，融资条件虽有退出风险，但我们不特别担忧。
2. **占用时长**：起初预计1982年1月付清。
3. **更好报价**：即使 KKR 退出，阿克塔管理层和董事会早已坚决出售，会有其他买家出现，但报价可能更低。
4. **失败后果**：红杉林索赔价值在**零到全部**之间，尽管巴菲特自称“分不清榆树和橡树”，但他冷静判断，倾向于**全体**赔偿的可能性更大。

#### 5.3 实际时间线与操作

| 日期          | 事件                                                         | 操作/影响                                        |
| ------------- | ------------------------------------------------------------ | ------------------------------------------------ |
| 1981.09.28    | 董事会原则同意出售给 KKR                                     | —                                                |
| 1981.09.30 起 | 开始买入，股价约 **33.50美元**                               | 八周内买入 **40万股**（5%股本）                  |
| 1981.12       | 结案推迟至次年1月4日                                         | 继续增持至 **65万股**（超7%），均价约 **38美元** |
| 1982.02.25    | 贷款人因房屋衰退“再次审视”交易，股东大会推迟至4月            | 发言人否认交易陷入困境                           |
| 1982.03.12    | KKR 最初方案作废，报价先削至33.50美元，后提至35.00美元       | —                                                |
| 1982.03.15    | 董事会拒绝 KKR 新报价，接受另一公司报价：**37.50美元 + 红杉林索赔额外所得的50%** | —                                                |
| 1982.06.04    | 股东收到 **37.50美元/股**                                    | 交易完成                                         |

#### 5.4 收益与后续惊喜

| 指标                   | 数值            |
| ---------------------- | --------------- |
| 总成本                 | **2,290万美元** |
| 总收入                 | **2,460万美元** |
| 持股时间               | 约 **6个月**    |
| 年化回报率（不含索赔） | **15%**         |

后续发展更令人惊喜：1987年1月，法院指定的委员会认定：

- 红杉林价值：**2.757亿美元**
- 利率：**复利、混合利率约14%**

尽管此案过程曲折，最终回报却远超预期。



### 六、伯克希尔的套利风格与行业对比

| 特征       | 伯克希尔                                 | 业内其他套利者               |
| ---------- | ---------------------------------------- | ---------------------------- |
| 参与频率   | **每年很少**，但单笔规模很大             | 每年可能进行50次甚至更多     |
| 决策依据   | **仅依据公开信息**，不凭谣言或猜测接管方 | 常利用各种信息渠道           |
| 生活模式   | 不整天盯盘，追求有意义的思考             | 需同时监控大量交易和行情波动 |
| 多元化程度 | 极为有限，单个交易对业绩影响显著         | 高度分散，单笔影响较小       |
| 时机选择   | 仅在**胜券在握**时出手                   | 经常随潮流而动               |

💡 **巴菲特哲学**：“如果只是为了赚钱而整天盯着股票行情，这种生活有什么意义可言？”



### 七、冷静期的思考与原则（1989年展望）

尽管1988年套利成绩出色，我们仍希望**冷眼旁观**：

- **长期仓位上升**，手中现金下降，无需为套利而套利
- **并购浪潮过于泛滥**，贷款和买家双方表现出放纵的乐观主义
- 我们不会预测股市、利率或商业活动短期走势

💡 **核心原则**：

> “别人处理他们的事情越不谨慎，我们处理自己的事情就要越谨慎。”

💡 **经济学家赫伯·斯坦的智慧**：

> “如果一些事情注定不能长久，那么终将会覆灭。”

套利交易中，当市场情绪过于亢奋，许多参与实际上已接近于**“谁是更傻的傻瓜”**的游戏。正如华尔街人士雷·迪福所言：“天使都害怕交易的地方，傻瓜却蜂拥而入。”

我们仍会参与套利，有时规模还会很大，但**仅仅在胜券在握的时候才会出手**。



## C.戳穿标准教条

### 一、对市场有效理论的批判

#### 1.1 理论主张与现实背离

市场有效理论（Efficient Market Theory, EMT）在20世纪70年代一度被奉为学术界的“圣经”。其核心观点认为，所有公开信息均已反映在股价中，市场总是无所不知，因此股票分析毫无用处——随意扔飞镖选出的组合可与最优秀分析师的表现相媲美。

巴菲特用长达63年的实践记录否定了这一论断：

> “以格雷厄姆-纽曼公司、巴菲特合伙公司、伯克希尔公司连续63年的套利经历而言，可以证明市场有效理论是多么愚蠢。”

| 时间段    | 机构                      | 去杠杆后套利年化回报率 |
| --------- | ------------------------- | ---------------------- |
| 1926–1956 | 格雷厄姆-纽曼公司         | 20%                    |
| 1956–1988 | 巴菲特合伙公司 / 伯克希尔 | 超过20%（未精确计算）  |

同期市场一般年化回报率（含分红）略低于10%。如果以1000美元为起点复利计算63年：
- 按 **10%** 增长 → **40.5万美元**
- 按 **20%** 增长 → **9700万美元**

这种量级差异足以激起任何理性人的好奇心，但市场有效理论的支持者却从未正视反面证据。

#### 1.2 公平测试的四个条件

巴菲特强调，上述优异业绩并非偶然，它满足严格的检验标准：

1. **样本广度**：在过去63年中，三个机构交易了数以百计不同类型的证券。
2. **稳定性**：最终结果没有被少数几次幸运经历所扭曲。
3. **信息对称**：仅依据公开事件进行套利，不需挖掘模糊事实或培养神秘洞察力。
4. **范围明确**：套利仓位有清晰界定，并非事后挑选。

#### 1.3 理论的顽固与投资者的优势

尽管市场有效理论已不如从前流行，但它在大学商学院中仍是必修内容，且那些曾经误导学生的支持者从未公开认错。对此，巴菲特略带讽刺地评论道：

> “从利己主义的观点出发，格雷厄姆的信徒们应该捐助大学里的教学职位，以确保市场有效理论的教学能一直持续下去。”

💡 **核心启示**：当大量市场参与者相信“努力无益”时，坚持独立判断与深度分析的投资者便获得了巨大的竞争优势。

⚠️ **重要警告**：套利并非总是高回报的保证。在很多时间里市场是有效的，许多套利机会因价格合理而被放弃。**投资者不可能仅凭一种特定方法或风格获得超额利润，只能通过认真评估事实并持续执行纪律来实现。**



### 二、长期持有杰出公司

#### 2.1 拒绝“剪除鲜花，浇灌杂草”

与那些急于卖出表现稍好公司、却坚定持有表现不佳公司的人截然相反，巴菲特主张永久持有优秀企业。

> “我们依然认为，如果一家公司既业务明晰，又持续保持优秀，那么出售这家公司的权益显然是愚蠢的行为——这种类型的公司简直难以取代。”

他以企业主的思维类比：当一家子公司的长期表现极其优异时，没有哪个CEO会愿意卖掉这颗“皇冠上的宝石”。然而，同样是这个人，在个人投资中却极易在经纪人鼓动下鲁莽地频繁买卖股票。

💡 **箴言**：**“在有利润的情况下，你就不可能破产”是错误的**——不能用这个理由去说服董事会出售明星子公司。

#### 2.2 可口可乐的世纪证据

| 年份     | 事件                                     | 投资结果                  |
| -------- | ---------------------------------------- | ------------------------- |
| 1919     | 可口可乐上市，股价40美元                 | 初始投资40美元            |
| 1920年末 | 市场悲观，股价跌至19.50美元（跌幅超50%） | 账面严重亏损              |
| 1938     | 已占据行业主导地位，年销2.07亿箱         | 分红再投资价值达3277美元  |
| 1993     | 年销107亿箱，相比1938年物理体量增长50倍  | 分红再投资价值超210万美元 |

即使在1938年——公司已成立五十多年、被《财富》杂志称为“发现太晚了”的时代，40美元的投资持有至1993年仍能变成约2.5万美元。

《财富》1938年的评价至今仍适用：“很难找到能与可口可乐在规模和销售上相匹敌，并且保持十年产品没有变化的公司。”

#### 2.3 集中投资的理念基础

> “芒格和我认为，在一生的投资生涯中，很难做出数以百计的明智决策。……我们采取的策略是，对自己仅仅要求次数有限的聪明，而不是过分聪明。我们现在每年只落实一个好主意。”

这一策略使伯克希尔免于陷入标准多元化的教条。巴菲特坚持：**集中投资于最了解、风险最低、利润潜力最大的公司，能够真正降低风险**——只要投资者在购买前对公司状况有足够深入的理解和满意。

**永久持股声明**：伯克希尔希望永久持有大都会/ABC公司、盖可保险公司和《华盛顿邮报》——即便股价涨到不可思议的高点也不出售，如同对待全资拥有的禧诗糖果一样。



### 三、风险的真谛

#### 3.1 模糊的正确胜过精确的错误

学术界以“贝塔”值（股价相对于市场的波动性）来定义风险，认为股价相对跌幅越大，风险就越大。巴菲特严厉批评这种定义：

> “在评估风险的时候，一个追求贝塔的纯粹主义者会不屑于调查公司的产品、竞争力如何或公司的负债情况……他所重视的不过是该公司股票的历史价格。”

这与真实商业世界的逻辑格格不入。他以1973年买入《华盛顿邮报》为例：股价越低时买入，按照贝塔理论“风险”竟变得更大——对一个能以巨大折扣买入整个公司的人而言，这毫无意义。

#### 3.2 真正的投资风险：五大衡量因素

对投资者而言，**真正的风险在于**：在预期的持股期间，税后总收益是否能至少保持在投资开始时的购买力，并加上合理的利率回报。评估时需考量以下因素：

| 序号 | 关键因素             | 说明                                             |
| ---- | -------------------- | ------------------------------------------------ |
| 1    | 长期经济特征的确定性 | 公司能否在漫长的未来中保持盈利模式不变？         |
| 2    | 管理层能力的确定性   | 包括充分实现企业潜力和明智运用现金流的能力       |
| 3    | 管理层是否回报股东   | 关注管理团队是将利润回馈给股东，还是优先自我利益 |
| 4    | 购买价格             | 安全边际至关重要                                 |
| 5    | 通货膨胀与税务水平   | 对购买力缩减的影响                               |

虽然这些因素无法从任何单一数据中精确计算，但就像大法官斯图尔特对“淫秽下流”所下的判词：“当我看见，我就知道。” 投资者同样可以对特定投资中固有的风险形成模糊却有用的判断。

#### 3.3 竞争优势胜过贝塔值

以可口可乐和吉列为例：
- 可口可乐占据全球软饮料市场44%的份额。
- 吉列占有全球刀片市场超过60%的份额（按价值计）。
- 两者市场份额多年来仍在提升，拥有强大的品牌力、产品属性与分销系统，构成宽阔的“护城河”。

然而，这两家公司的贝塔值却与许多平庸企业相差无几。贝塔无法区分一家销售宠物石头和一家拥有芭比娃娃的玩具公司的内在风险——但任何具备适度消费行为理解能力的投资者都能轻松分辨。

💡 **实用准则**：将自己限制在易于理解的行业里，勤奋而理智的人能够以相当精确度判断风险。对于科技快速变化的行业，无法可靠评估长期前景，则应坚守在能力圈之内。



### 四、多元化与集中投资的辩证

#### 4.1 何时需要多元化

| 场景                   | 策略                                                         |
| ---------------------- | ------------------------------------------------------------ |
| **套利活动**           | 广泛多元化，通过多个相互独立的交易降低单笔风险，用概率优势获利 |
| **对具体公司不甚了解** | 持有大量股票并分批买入，例如定期投资指数基金                 |
| **具有深入知识储备**   | 集中投资于5至10家价格合理的优质公司，分散反而拉低收益、提高风险 |

> “当一个‘傻’钱承认自己的局限，它就不再傻了。”

#### 4.2 集中投资的逻辑

如果一位投资者能识别出最优秀的公司名单，那么将资金投到排在第20名而非首选前5名，显然不合逻辑。正如梅伊·韦斯特所言：**“太多好事过了头反而成了坏事，这可能才是最精彩的地方。”**

在奥马哈的非上市公司中，巴菲特会评估每一家公司的长期经济特征和管理层素质，然后以合理价格集中买入运营最为良好的几家。面对股市中的上市公司，没有理由采取不同方式。

#### 4.3 凯恩斯的智慧

经济学家凯恩斯早于1934年便在信中写道：

> “随着时光的流逝，我越来越感到确信，投资的正确方式是将相当分量的资金，投资于你了解并拥有令人充分信任的管理层的公司。一个人的知识和经验注定是有限的，在某些特定的时间段，使我感受到充分信心的公司很少超过两或三家。”



### 五、市场行为的深刻反思

#### 5.1 “投资组合保险”的荒谬逻辑

1986–1987年流行的投资组合保险策略，本质上就是高级版的止损单：股价越跌，越要自动抛售。它导致1987年10月约有600亿至900亿美元的股票处于连锁抛售触发状态。

> “如果你认为投资顾问是被雇来帮你进行投资的，那么你可能会被各种技术术语弄得晕头转向。……当一个农场主看到隔壁农场被低价贱卖时，会要求自己的农场也以低价出售吗？”

这种非理性行为正是机构投资者“多动症”的极端表现——投资经理们关注的是其他基金经理明天会干什么，而不是企业未来几年会怎样。

#### 5.2 波动是真正的投资者之友

巴菲特坚定地认为：**股市是一个重新配置财富的地方，钱会从活跃者手中流向耐心者手中。** 大机构造成的市场波动，恰恰为坚守投资体系的理性投资者创造了更多明智买入的机会。只有那些因财务或心理压力而被迫在市场艰难时卖出的人，才会被波动所伤害。

> “如果芒格和我认为，自己可以通过在不同花朵之间跳来跳去的方式取得长期成功，那简直是天方夜谭。我们认为，将那些频繁买卖的机构称为‘投资者’，就像将喜欢一夜情的人称为浪漫主义者一样可笑。”



## D.“价值”投资：多余的两个字

### 一、核心投资原则：寻找杰出公司

#### 1.1 理念要旨
我们的目标是发现那些**价格合理的杰出公司**，而不是价格便宜的平庸公司。  
> 正如芒格和我学到的：巧妇难为无米之炊。我们能用丝绸织出丝绸钱包，但前提是先拿到丝绸。

#### 1.2 亲身教训
早年寻找“便宜货”的经历让我遭受惩罚、受到教育：

| 失败投资类型   | 举例                             | 教训         |
| -------------- | -------------------------------- | ------------ |
| 短线农具生产厂 | 三流百货公司、新英格兰地区纺织厂 | 远离平庸公司 |

💡 **关键启示**：购买好企业至关重要。我也曾为此付出20年才彻底领悟的代价。

#### 1.3 管理层：德才兼备
我们不仅买好公司，更要求管理层**诚实且能干**。  如果我们遇上志不同、道不合的管理层，控制权赋予我们更换的权力，但这个过程如同婚姻的改变——**痛苦、耗时且充满风险**。

**优秀管理者典范：**
- 大都会公司：汤姆·墨菲与丹·伯克
- 盖可保险：比尔·斯奈德与卢·辛普森
- 《华盛顿邮报》：凯·格雷厄姆与迪克·西蒙斯

> 没有他们，我们不可能做得更好。



### 二、控制类投资 vs 非控制类投资

控制类投资具有两个主要优势：

#### 2.1 两大优势对比

| 优势维度       | 控制类投资（持股80%以上）                     | 非控制类投资（可流通证券）               |
| -------------- | --------------------------------------------- | ---------------------------------------- |
| **资产配置权** | 可直接支配资产配置，避免CEO配置能力不足的损耗 | 无话语权，依赖管理层配置能力             |
| **税务成本**   | 可向母公司输送更多利润（新税法下优势显著）    | 吸收较多税务成本，尤其受1986年新税法影响 |

##### 关于资产配置的深层问题
- 许多CEO在上任前从未做过资产配置决策，其难度堪比让一位杰出的音乐家突然去担任美联储主席。
- 若一家公司每年留存利润相当于净资产的10%，十年后CEO需合理运用公司60%以上的资本。
- 结果往往催生了大量**不明智的配置**，这也是“公司重组”新闻频现的原因。

#### 2.2 非控制类投资的独特抵消优势
市场先生偶尔会以**极低的可笑价格**出售杰出公司的非控制部分权益，这远低于控股权转让的谈判价格。  **经典案例：**

| 公司           | 买入时间与价格               | 后期运营利润（税后/股） | 回报特征           |
| -------------- | ---------------------------- | ----------------------- | ------------------ |
| 《华盛顿邮报》 | 1973年，5.63美元/股          | 1987年达10.30美元       | 长期价值显著低估   |
| 盖可保险       | 1976~1980年，均价6.67美元/股 | 报告年达9.01美元        | 同样以极低价格买入 |



### 三、权益类投资策略详解

#### 3.1 选股四项铁律
我们的方法自1977年以来本质未变，仅将“有吸引力的价格”强化为“非常**有吸引力的价格**”。

| 序号 | 特征要求                       |
| ---- | ------------------------------ |
| ①    | **我们能理解**                 |
| ②    | **具有良好的长期前景**         |
| ③    | **具有诚实且能干的管理层**     |
| ④    | **能以非常有吸引力的价格买到** |

#### 3.2 “价值”与“成长”并非对立
我们拒绝将二者割裂的模糊思维。**成长本身就是价值计算的组成部分**，其影响从微小到巨大，可以是正面因素也可以是负面因素。

> “价值投资”这一术语实属多余。任何买入行为若不是在寻找物有所值的机会，就不是投资。有意识地为股票支付高价并希望以更高价迅速卖出，应贴上**投机**标签——虽不违法，但无法赚大钱。

#### 3.3 成长何时有害？
⚠️ **重要警示：** 只有当每投入一美元资本能创造出超过一美元的市场价值时，成长才对投资者有利。在低边际回报率的行业，成长反而**损害股东利益**，比如美国航空业——行业越扩大，投资者灾难越深重。

#### 3.4 价值方程式与安全边际
**约翰·伯尔·威廉姆斯的定义（精简版）：**

> 任何股票、债券或公司的今日价值，等于其预期存续期内，以合适利率贴现后的现金流入与流出。

- 债券现金流入确定性强，但管理层无能和诚信问题可能导致暂停付息。
- 权益“票息”则**极大倚赖管理层品质**。

最终结论：  最值得拥有的公司是那些能**在漫长周期中，持续利用增量资本获取高回报率**的企业。  

💡 **基石原则**：我们坚守格雷厄姆提倡的**安全边际**，股价仅比价值略高时我们毫无兴趣。



### 四、案例精研：投资大都会/ABC公司

#### 4.1 投资背景与信心表态
- **行动**：年底以172.50美元/股购买300万股，与1985年3月市价持平。
- **评价**：大都会管理层（汤姆·墨菲与丹·伯克）为全美最佳，不但是杰出经理人，也是你愿意将女儿嫁给他的那种人。
- **心态**：协助大都会筹集35亿美元收购ABC。即使ABC短期经济前景平淡，我们**极具耐心**。  
#### 4.2 独具匠心的安排
为表信心，芒格与我主动提议**投票权代为行使**及**持股出售限制**：

| 安排内容                                   | 目的与效果                                   |
| ------------------------------------------ | -------------------------------------------- |
| 只要汤姆（或丹）担任CEO，投票权由他们行使  | 强化管理层主导，确保长期战略稳定             |
| 未经管理层许可，不得将持股出售给任何大买家 | 防止“旋转门式资本家”的短期掠夺，稳定公司治理 |

#### 4.3 经济与人性的双重回报
- **经济层面**：稳定将优秀管理层与杰出资产结合，为丰厚财务回报培育土壤，大幅提升了成功的确定性。
- **人性层面**：我们不让尊重和喜欢的管理层因持股变动而夜不能寐。我们已承诺“不会有意外”，且伯克希尔会把承诺信守到底——即便我个人（巴菲特）活不到三位数年纪。

#### 4.4 巴菲特的幽默自省
💡 读者们或许会问：为何1978~1980年我以43美元/股清仓大都会，如今又以172.50美元/股买回？  我已经在1985年花费大量时间寻找漂亮的借口来抹平这些矛盾……  请再给我一点时间吧。

## E.聪明的投资

### 一、 股权思维与长期主义
#### 1.1 像经营企业一样投资
买入一家公司部分股权（普通股）与收购整家公司，运用的艺术并无二致。  
你所需要考虑的仅有：**标价合理、质地优良的公司，以及能干且诚实的管理层**。  
在此之后，只需持续观察这些品质是否得以保持，而不是因为预期利率波动或华尔街观点变化就出售那些盈利颇佳的公司。

#### 1.2 集中投资的智慧：不要卖掉你的“乔丹”
遵循上述理念，投资组合自然会集中在少数几只股票上——好比拥有一支杰出大学篮球队20%的权益，仅少数球员就能进入NBA成为明星，贡献绝大部分收益。

> 💡 **卖出组合中获益最大的股票，只因它占比过重，这种做法好比公牛队因为迈克尔·乔丹得分太多而将他卖掉，实在愚蠢。**



### 二、 选股之“锚”：寻找确定性
#### 2.1 拥抱不变，远离剧变
我们寻找的是在未来10年、20年几乎可以肯定依然具有竞争优势的公司。  
一个环境迅速变化的行业或许能提供巨大的胜利机会，但它**不具备我们追寻的确定性**。

> 作为公民，我们欢迎变革、创新和生活水平的提升；作为投资者，我们对迅速膨胀、变化的行业的态度正如对太空探索：**我们会鼓掌欢迎，但不会参与其中。**

#### 2.2 案例：不变的核心基因
##### 禧诗糖果

| 维度      | 发生了变化的                     | 从未改变（1920年至今，且未来20~50年亦然） |
| --------- | -------------------------------- | ----------------------------------------- |
| 产品/设备 | 提供不同类型的糖果，使用不同机器 | 人们购买盒装巧克力的根本理由              |
| 渠道      | 通过不同分销渠道销售             | 为什么在众多选择中仍选择我们              |

##### 可口可乐
- 可口可乐的**基本面因素**——可乐的**竞争优势**与**出色经济特质**——历经百年依然稳定。  
- 1896年（问世仅10年）的年报已绘就未来100年的蓝图：  
  - 当年销售额14.8万美元，汽水销量116,492加仑；1996年销量达3.2亿加仑。  
  - 总裁阿萨·坎德勒的话至今仍是公司基石：“我们向世界宣告，可口可乐对于所有人而言，是一种卓越的健康产品以及美好的感觉。”  
  - 销售力量：从十名外勤销售员起步，如今已覆盖全球。

#### 2.3 “注定如此成功”的幻觉
- 芒格和巴菲特终其一生也只能鉴别出**极少数的**“注定如此成功”的伟大公司。通用汽车、IBM、西尔斯百货等长期看似无敌的公司，最终证明并非如此。  
- 因此，我们拿不出“漂亮50”或“闪亮20”的名单，只能在组合里标出几个**“极高可能性”**的公司。

⚠️ **追高的代价**  
即便对最杰出的公司，**支付过高价格**也会带来风险——可能需要很长时间才能产出与价格匹配的价值。当市场过热时，这一点尤其值得警惕。

#### 2.4 警惕管理层失去专注
当一家杰出公司的管理层无视核心业务，去收购平庸的企业时，投资者的痛苦将极为漫长。  
- 真实案例：可口可乐养过虾，吉列搞过石油。  
- 💡 失去专注是巴菲特和芒格最为担心的风险。好在现有及未来的管理层不会在可口可乐和吉列再犯此类错误。



### 三、 投资者操作指南
#### 3.1 大多数人的最佳选择：指数基金
大多数投资者（机构与个人）会发现，持有**低成本的指数基金**是投资股票最好的方法。以这种方式，可以注定战胜绝大多数专业投资者的净回报。  
- 投资者本可以轻松搭乘美国公司这趟经济快车，但很多人偏偏选择频繁交易。

#### 3.2 主动投资者的必修课
若你打算自己构建投资组合，请记住：

- 你需要正确评估**能力圈内**的企业。圈子的边界大小不重要，**清楚知道边界才关键**。  
- **抛弃**贝塔、有效市场理论、现代投资组合理论等过时教条。只需学好两门功课：  
  1. 如何评估一家公司的价值  
  2. 如何对待市场价格  
- **投资目标**：以理性价格，买入那些简单易懂、盈利在未来5~20年确定能大幅增长的企业。  
  - 符合条件的公司少之又少，一旦出现，应**大幅买入**，而非小赌怡情。  
- **持有纪律**：如果不打算持有一只股票10年，那就连10分钟都不必考虑持有。

#### 3.3 价值增长的本源：透视盈余
伯克希尔市值的增长，源于我们所投企业**盈利总额**的持续提升。如果是这些公司的盈利没有着落，那么伯克希尔的价值也就不会增加。

#### 3.4 投资者表现不佳的三大原因

| 原因         | 具体表现                                           |
| ------------ | -------------------------------------------------- |
| **高成本**   | 交易过于频繁，或支付过多的管理费                   |
| **跟风决策** | 以股评、市场时尚为依据，而非深思熟虑的企业定量分析 |
| **择时错误** | 常常牛市进入，熊市退出                             |

> 投资者应牢记：**亢奋与成本是敌人**。如果想尝试择时，应在别人贪婪时恐惧，在别人恐惧时贪婪。



### 四、 能力圈与市场纪律
#### 4.1 坚守能力圈，坦然说“不知道”
- 对于某种可持续的重要竞争力，我们偶尔能识别，更多时候无法高度自信地做到。因此，我们**不买高科技公司的股票**——尽管我们认同它们的创新改变了世界，但对其中哪些参与者拥有真正持久的竞争力，我们毫无洞见，且无法通过学习解决。  
- 同样，专利评估、制造过程、地质勘探等也非我们专长，我们对此不发表意见。  
- 💡 我们坚守所了解的东西。在已建立的能力圈内，机会永远都存在。

#### 4.2 不预测市场，只谈现实预期
- 我们从未试图预测股市下个月或明年的走势。当股票价格高高在上时，我们更喜欢企业本身，但不会大幅削减投资。  
- 若有两种公司供选择——**“价格合适但运营有问题”与“价格有问题但运营良好”**，我们倾向后者。当然，最吸引我们的是**价格合适且运营良好**的公司。  
- 公司利润增长与GDP名义增长（实际约3% + 通胀约2% ≈ 5%）大致相当。加上分红，股票的长期总体回报不太可能远高于此。一旦投资者预期回归现实，投机板块可能面临严峻调整。  
- 伯克希尔终将再度大举投资股市，但“有谁知道在什么地方？或什么时候？”

#### 4.3 泰德·威廉姆斯的击球纪律
在《击球的科学》中，泰德将击球区划分为77个格子，只挥击落在“最佳”位置的好球才能打出0.400的巨星成绩；挥击“最差”球则成绩跌至0.230，与名人堂失之交臂。

| 棒球比喻                 | 投资映射                         |
| ------------------------ | -------------------------------- |
| 77个格子的击球区         | 纷繁而至的投资机会               |
| 只打“最佳”格子里的球     | 仅投资进入理想价位的杰出企业     |
| 落在下端外侧角的差球     | 当前吸引力不足的平庸机会         |
| 可放过差球，不会被罚出场 | 投资者可以耐心等待，没有三振出局 |

> 唯一不同：我们不会因放过三个差球而出局，但日复一日扛着球棒等待完美一击，也绝非易事。当有人信誓旦旦解释市场的“疯狂魔法”时，别忘了：“**傻瓜才会给你理由，智者从来不会。**”



## F.捡烟蒂和惯性驱使（机构强迫症）

### 一、 两大投资策略反思

#### 1.1 “捡烟蒂”投资法的陷阱

“捡烟蒂”投资法指以极低价格买入资质不佳的公司，期待价值波动带来获利机会。这种方法存在根本性缺陷。

| 陷阱类型     | 具体描述                                                     |
| :----------- | :----------------------------------------------------------- |
| **表面便宜** | 问题公司往往“祸不单行”，最初看似便宜的价格，最终可能因持续涌现的问题而并不便宜。 |
| **价值侵蚀** | 即便最终能以略高于成本价的价格出售，但若持有时间过长，公司的低回报会严重侵蚀初期优势，导致整体回报令人失望。 |
| **结论**     | 除非你是专业清算师，可立刻处置资产，否则此法愚蠢。           |

> 💡 **核心洞见：时间是优秀企业的朋友，是平庸企业的敌人。**

#### 1.2 避开难题而非解决难题

经历多次失败后，我们认识到投资成功的关键在于回避问题，而非解决问题。

> 💡 **核心理念：优秀的骑手只有在良马上，才会有出色的表现，在劣马上会毫无作为。**

- **管理层与企业的关系**：同样能干、忠诚的管理层，在具有优秀经济特征的公司里会取得优良成绩；但在流沙般的企业中则难有进展。当一个声誉卓著的管理者接手一家经济状况不佳的公司时，能保全的往往只有公司的虚名。
- **自我能力认知**：我们并未学会如何解决困难企业的问题，所学到的是 **避开它们**。我们专注于寻找可以跨越的`1英尺跨栏`，而非挑战`7英尺跨栏`。
- **应对策略**：通常，坚守于易于明白且显而易见的目标，比摆脱困境利润更丰厚。伟大的投资机会偶尔会出现在优秀公司遭遇一次性、可解决的问题时，但总体上，我们倾向于 **避开怪兽，而不是杀死它**。



### 二、 企业内部的隐形敌人：惯性驱使

这是一种足以压倒一切的、看不见的力量，我们称其为“惯性驱使”或“机构强迫症”。当这种强迫症发作时，理性经常会枯萎。

| 惯性驱使的表现                                               |
| :----------------------------------------------------------- |
| 1. 机构会抗拒在当前方向上的任何改变，如同受牛顿第一运动定律控制。 |
| 2. 公司的项目或收购行为会像工作填满时间一样，耗尽所有现金。  |
| 3. 公司领导人所渴望的项目（无论多愚蠢），总会得到下属精心准备的回报论证和战略分析支持。 |
| 4. 同行的行为（扩张、收购、管理层薪酬等）都会被不加思考地模仿。 |

> ⚠️ **注意**：令公司误入歧途的，是惯性的力量，而非腐败或愚蠢。我们一直努力将这种影响降至最低，并将投资集中于那些对此问题保持警惕的公司。



### 三、 核心投资与人际原则

#### 3.1 与对的人同行

我学会了只与那些我喜欢、信任和尊敬的人做生意。

- **原则本质**：这条原则本身不能保证成功——二流的工厂不会仅因管理层人品好而繁荣。
- **奇迹创造**：一个股东若能将自己与经济特征不错的公司中的优秀人士结合，也能成就奇迹。
- **反面教训**：**与不值得尊敬的经理人为伍，无论公司前景多么诱人，我们从未成功过。**

#### 3.2 警惕疏忽之错

我所犯过的一些最糟糕的错误，是错过了那些完全在我能力圈范围内、送上门来的真正大机会。这种错误的代价极其巨大。

#### 3.3 坚守保守财务

- **决策分析**：如果1965年使用传统意义的高杠杆，我们几乎（99%概率）可获得远高于已实现的23.8%的年回报率，仅有1%的风险可能导致违约。
- **核心立场**：
  - **我们不喜欢这种`99:1`的发生概率，永远不会。** 一个小的丢脸或痛苦，不能被一个大的额外回报所抵消。
  - 如果你的行动是合理的，你肯定会得到良好的结果。在绝大多数情况下，杠杆只会加速事情的运动。
  - 芒格和我从不匆忙，**我们享受投资过程远胜于收获的结果**。



## G.生命与负债

#### 一、 债务使用的核心纪律

我们极少使用债务，仅在特定、可控的场景下才会破例。所有债务的使用均需满足严格的前提条件。

| 目的分类           | 具体描述                                                     | 风险控制原则                                               |
| :----------------- | :----------------------------------------------------------- | :--------------------------------------------------------- |
| **短期投资策略**   | 偶尔参与美国政府或其代理机构为主体的回购交易。               | 仅作为短期策略组成部分。                                   |
| **对冲投资组合**   | 为对冲投资组合中的付息应收账款而借款。                       | 💡 **前提**：对所涉品种的风险特性了如指掌。                 |
| **子公司独立负债** | 子公司（如中美能源）自身报表上的负债，可能并入伯克希尔合并报表。 | ⚠️ **关键**：伯克希尔公司层面**不对这些债务提供任何担保**。 |



#### 二、 风险视角：集中与分散

从风险角度看，盈利来源的多元化远比单一的保守负债结构更重要。

| 对比维度       | 单一公用事业                       | 多元化公用事业组合（如中美能源）               |
| :------------- | :--------------------------------- | :--------------------------------------------- |
| **假设条件**   | 负债结构非常保守                   | 盈利覆盖利息比例为 **2:1**                     |
| **风险特征**   | 一次灾难性事件足以导致其资不抵债。 | 一次区域性灾难（如西部地震）不会构成致命打击。 |
| **安全性结论** | 脆弱                               | **更为安全**                                   |

> 中美能源公司正是通过其不断扩大的、多元化的管制盈利，很好地运用了其负债。



#### 三、 伯克希尔层面的最高原则：坚决抵制重大负债

在伯克希尔集团层面，我们对于以重大负债形式进行收购或企业运营**毫无兴趣**。

##### 3.1 核心考量因素

我们抵制高杠杆的原因根植于对股东和客户的深刻责任。

- **对股东的责任**：
    - 数以千计的股东将大部分家产托付于我们的股票。
    - 绝大部分公司董事和管理层也在此股东名单上。
    - **任何对公司的灾难，就是对他们个人的灾难。**

- **对保险客户的承诺**：
    - 对于遭受永久性损伤的投保客户，承保支付可能延续**50年**或更久。
    - 我们曾给予完全安全的承诺，并会信守诺言，无论发生何种极端情况：
        - 金融危机
        - 股市关门（如1914年）
        - 美国本土遭受核弹、化学武器或生化武器攻击。

##### 3.2 风险抵御能力的基础

我们愿意承担其他保险公司无法企及的巨大风险（如高额单次重灾承保），这份底气源于我们坚实的基本面。无论市场如何剧烈波动（如1987年10月股灾），伯克希尔都确信拥有充足的：

-   **净资产**
-   **盈利现金流**
-   **可变现性**



#### 四、 关于杠杆的根本警示：数字与零的哲学

历史反复证明了一条残酷的数学真理，无论使用者多么聪明，杠杆最终的结果通常是归零。

> “无论多么长的一串数字，只要乘上一个零，其结果一定等于零。”

##### 4.1 杠杆的双刃剑效应与成瘾性

杠杆会放大收益，带来短暂的辉煌，但它也具有极高的成瘾性。

-   **阶段一：获利时**
    -   你的配偶会认为你**很聪明**。
    -   你的邻居会**羡慕**你。
-   **阶段二：成瘾后**
    -   一旦奇迹般获利，很少有人愿意再回到保守状态。
-   **阶段三：归零时**
    -   正如2008年金融危机再次教育我们的：任何序列的正数，乘以一个零，都会蒸发殆尽，一切归零。

##### 4.2 企业负债的致命风险：再融资假设

拥有巨额负债的公司通常会假设到期债务总能得到再融资。这个假设通常是有效的，但一旦失效，将生死攸关。

-   **失效情形**：
    1.  企业内部的特别原因。
    2.  市场正常的资金短缺。
-   **最终结果**：债务到期必须即刻偿还，不得延展。此时，**只有现金才能解决问题**。



#### 五、 终极隐喻：信用就像氧气

信用与氧气有着惊人的相似性，其重要性在消失时方才凸显。

| 状态       | 对氧气/信用的感知          | 后果                               |
| :--------- | :------------------------- | :--------------------------------- |
| **充足时** | 人们不会注意到它们的存在。 | 习以为常，一切运转正常。           |
| **消失时** | 人们才会发现它们的重要性。 | 信用的短暂缺失都会让公司陷入困境。 |

> **例证**：2008年9月的危机中，国民经济中很多部门的信用在一夜之间消失，令整个国家陷入崩溃的危险边缘。
