# 第1章 公司治理

## A. 完整公平的信息披露

> **核心原则**：在伯克希尔公司，我们通常会**换位思考**，设想如果我们自己处于股东的位置，应该得到什么样的信息，我们会从这样的角度向股东披露完整的报告。

### 一、信息披露的核心理念

芒格和我会公开当前状况下，公司的**所有重要的运营状况**，以及公司高管对于**长期商业经济特征的坦诚看法**。无论是很多财务细节，还是任何重大的运营数据的讨论，我们都希望给出**解读**。

- **我们反对的内容**：
  - ❌ 人物、工厂、产品的照片
  - ❌ 引用 **EBITDA**（未计利息、税项、折旧及摊销前的利润）的概念
  - ❌ 模糊不清的会计技巧
  - ❌ 由公司公关部或财经公关顾问发出的信息

- **我们期待的内容**：
  - ✅ 公司CEO能**直接用自己的语言**解释所发生的事情
  - ✅ 坦诚、清晰、无修饰的运营状况说明

> 💡 **技巧**：巴菲特和芒格认为，引用EBITDA的概念会让他们"不寒而栗"——难道管理层认为公司的资本开支会由牙齿仙女（Tooth Fairy）承担吗？



### 二、公平报告的定义

对于我们而言，所谓**"公平报告"**意味着：

| 要素         | 具体做法                                                     |
| ------------ | ------------------------------------------------------------ |
| **平等获取** | 我们的30万个"合伙人"（指伯克希尔的股东）能够在**同一时间**或尽可能同时得到信息 |
| **发布时间** | 选择在**周五股市收盘后**，或**次日的早晨**，将年度和季度财报发布在互联网上 |
| **消化时间** | 让股东或有兴趣的潜在投资者有充分的时间，在**周一开盘之前**消化信息 |



### 三、反对选择性披露

⚠️ **注意**：我们要为美国证券交易委员会（SEC）主席**小阿瑟·莱维特（Arthur Levitt, Jr.）**进行的股市打假活动鼓掌。

| 问题           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| **选择性披露** | 像癌症一样蔓延，已成为大型公司的标准化行为                   |
| **操作方式**   | 用于"指引"分析师或大股东对于盈利的预期，提供的数据会正好或略低于公司真实预期 |
| **本质**       | 这就像抛媚眼或点头一样，给那些满脑子投机思维的机构和顾问发出暗示 |
| **危害**       | 相对于以投资为导向的个人投资者，这些人得到了更多的信息优势   |
| **性质**       | 这实际上是一种**贪污腐败行为**，但却不幸地被华尔街和美国企业界奉为圭臬 |

> 感谢莱维特主席代表投资人利益而进行的不屈不挠且卓有成效的努力。现在，所有公司都被要求平等地对待他们的股东。但这种变革，不是出于道德良心，而是出于强迫压力，这对于公司CEO们以及公司的投资者关系部门而言，并不是一件光彩的事。



### 四、CEO盈利预测的警示

芒格和我们都认为，**CEO们预测公司增长速度，是骗人的和危险的**。当然，他们是被分析师和公司自己的投资者关系部门怂恿裹挟着这么做。但他们应该拒绝，因为这样预测多了会招致麻烦。

**为什么盈利预测是危险的？**

1. **不切实际的增长预期**
   - 只有非常少的大型公司能维持15%的年复合增长率
   - 测试：假设从1970年到1980年选出200家盈利最高的公司，看看有多少家可以保持15%的每股盈利增长率——**很少有公司可以满足这个标准**
   - 巴菲特打赌：2000年最赚钱的200家公司中，能在未来20年中保持15%每股盈利增长的公司，将**不超过10家**

2. **腐蚀CEO行为**
   - 许多CEO们参与那些并不经济的企业运作，仅仅是为了追求他们曾经宣布的盈利目标
   - 在使出重重解数、筋疲力尽之后，他们有时甚至玩起了各种各样的会计游戏，以"制造数字"，达成目标

3. **滚雪球效应**
   - 一旦公司将盈利从一个时期转移到另一个时期，运营的缺口将会随之在未来要求更多的会计操纵
   - 更加肆无忌惮，愚蠢就变成了欺诈
   - > "据悉，更多的钱是被'笔'偷走的，而不是被'枪'劫走的。"



### 五、给投资者的三个建议

| 序号  | 警示信号                   | 具体说明                                                     |
| ----- | -------------------------- | ------------------------------------------------------------ |
| **1** | ⚠️ **小心展示弱会计的公司** | 如果一个公司没有费用选项，或养老金假设如天马行空，对于这样的公司要小心。当公司管理层在公开的方面使用低下的手段，那么他们在背后可能采用同样的手法。**厨房里如果有蟑螂，绝不可能只有一只。** |
|       |                            | **鼓吹EBITDA是一种有害的做法**。考虑到折旧只是一种"非现金"成本，所以折旧并不是一项真实发生的费用——这样的计算毫无意义。事实上，折旧是一项特别没有吸引力的费用，因为现金在所获得的资产产生收益之前就已经支出了。 |
|       |                            | 💡 **比喻**：设想一下，今年伊始，有家公司一次性给员工发了未来十年的工资（这样，他们会编制一个十年的固定资产摊销表）。在接下来的九年里，这种预支的成本会是一种"非现金"成本——今年年初预付成本资产的减少。会有人在意，第二年到第十年的成本记录仅仅是一个简单的记账手续吗？ |
| **2** | ⚠️ **不知所云的注脚**       | 如果你看不懂财务报告中的某个注释或其他管理解释，这通常是因为**CEO们不想让你明白**。在安然公司欺诈案例中，对于某些交易的描述依然困扰着巴菲特。 |
| **3** | ⚠️ **大肆鼓吹盈利成长预测** | 企业很少能在波澜不惊、毫无意外的环境中一帆风顺地发展，所有盈利也不是简单顺利地可以预知。 |



### 六、伯克希尔的坦诚

芒格和我**不仅不知道伯克希尔下一年的盈利水平，甚至不知道下一个季度的盈利如何**。

> 对于那些一贯声称能够预知未来的CEO们，我们抱怀疑态度。如果他们总是能达到他们预测的目标，我们对其会是**彻头彻尾的怀疑**。那些习惯于"制造数字"的公司高管们，在某种程度上是试图粉饰数字。

## B. 董事会与公司高管

### 一、CEO与基层员工的考核差异

#### 1.1 基层岗位：清晰可量化的标准

基层职位通常具有**清晰的标准**和**可量化的表现指标**，不达标即出局：

| 岗位类型 | 考核标准                | 不达标后果   |
| -------- | ----------------------- | ------------ |
| 秘书     | 打字速度（如80字/分钟） | 立即失去工作 |
| 销售人员 | 销售目标                | 立刻被解聘   |

#### 1.2 CEO岗位：标准缺失与监督缺位

与基层员工形成鲜明对比，**CEO的考核存在系统性缺陷**：

- ⚠️ **衡量标准几乎不存在**：碌碌无为的CEO经常可以蒙混过关
- ⚠️ **缺乏即刻的监督考察机制**：即使公司运营败象毕露，CEO仍可推诿塞责
- **"先射箭后画靶"现象**：匆忙将落箭之点圈为靶心，事后合理化失败

#### 1.3 职位越高，跌得越轻

| 层级           | 监督强度 | 问责后果           | 利益保障                         |
| -------------- | -------- | ------------------ | -------------------------------- |
| 基层员工       | 高       | 立即解雇           | 无                               |
| 中层管理者     | 中       | 影响个人利益       | 有限                             |
| CEO/董事会成员 | 低       | 含混其辞、推诿塞责 | 即使公司被收购接管，仍获巨额利益 |

> ⚠️ **讽刺现象**：在董事会议上，对CEO成绩表现的批评就像打嗝一样无关痛痒，而不称职的打字员却无法享受这种待遇。



### 二、董事会里的三种情形

巴菲特指出，上市公司中存在三种根本不同的治理情形，尽管董事们的法律责任完全相同：

#### 2.1 三种情形对比

| 情形  | 股权结构特征           | 典型案例         | 董事会角色                             | 有效性                                             |
| ----- | ---------------------- | ---------------- | -------------------------------------- | -------------------------------------------------- |
| **1** | 无控股股东             | 大多数上市公司   | 应像缺席的股东一样，着眼于公司长期利益 | ⚠️ 容易被"长期"一词滥用，董事缺乏诚信或独立思考能力 |
| **2** | 控股股东同时是公司高管 | 伯克希尔公司     | 无须扮演中间人角色，除反对外别无他法   | 取决于控股股东品质                                 |
| **3** | 控股股东不参与公司管理 | 好时食品、道琼斯 | 外部董事可向大股东直接反映不满         | ✅ **逻辑上最为有效**                               |

#### 2.2 第一种情形的治理建议

> 💡 **理想董事会构成**：
> - 成员人数：**十个或更少**
> - 多数应来自**公司外部**
> - 外部董事应**制定CEO业绩衡量标准**
> - 定期召开**没有CEO参与**的会议，以衡量标准评判其业绩

**董事会成员必备特质**：
1. 精通业务
2. 乐于本职
3. 具有**股东利益导向**精神

> ⚠️ **错误做法**：基于社会名声或多元化因素挑选董事。和和气气但无所事事的董事一旦被任命，再无丢失工作之虞。

#### 2.3 第二种情形下的外部董事选择

如果股东/公司高管是平庸或糟糕之辈，董事除反对外无能为力。

> ⚠️ **关键行动**：如果情况没有改观，问题十分严重，**外部董事应该辞职**。他们的辞职将是对管理质疑的警示信号，强调已没有外部人士可以更正股东或公司高管的缺点。

#### 2.4 第三种情形的优势

- 大股东既不用自我批评，也不用麻烦地面临争取多数董事同意
- 可保证挑选的外部董事能给董事会带来积极有益的因素
- 外部董事知道其建议能被认真倾听
- 若大股东聪明且自信，会做出既尊重管理层又维护股东利益的英明决策



### 三、"独立"董事的真相

#### 3.1 当前"独立"董事的呼声与实质

> 💡 **巴菲特的支持**：由外部董事定期召开没有CEO参与的董事会，这种形式是一种改革，我对此持**热烈支持态度**。

**真正的"独立"董事应具备**：
- 能思考独立、观点独立
- **必须精通商务**
- 有兴趣并具有**股东利益导向**的特质

#### 3.2 "独立"董事的失败案例：投资公司董事会

**历时62年的案例研究（1942年以来）**：

| 职责         | 应然表现     | 实然表现                                     |
| ------------ | ------------ | -------------------------------------------- |
| 选择投资经理 | 寻找最佳人选 | 僵化刻板：A基金永远选A经理，B基金永远选B经理 |
| 谈判管理费用 | 争取最低费用 | 董事们毫不关心，降费对基金经理影响重大       |

> ⚠️ **讽刺对比**：当董事们处理自己的金钱时，他们会另辟蹊径、谨慎小心；肩负其他投资人托付时，心中却不这么想。

#### 3.3 并购案中的虚伪体现

在A公司被高价卖给B经理的案例中：
- "独立"董事们开始自我反省，认为B经理才是最佳人选
- B经理买下A公司后，身价即刻飞涨（因需补偿收购成本，从A公司股东口袋里捞回）
- 参考：约翰·博格《公募基金的常识》

#### 3.4 伯克希尔董事会的典范标准

以《圣经》"你的财富在哪里，你的心就在哪里"为标准：

| 标准     | 伯克希尔实践                                      |
| -------- | ------------------------------------------------- |
| 持股要求 | 每位董事至少将**400万美元以上**的身家放在伯克希尔 |
| 持股来源 | 没有任何股份是靠股票期权或赠予获得                |
| 酬劳水平 | 相较于自身年收入极其有限                          |
| 风险承担 | 没有替董事安排任何责任保险                        |

> 💡 **核心结论**：芒格和我已看过很多符合《圣经》"财富"观点的行为。根据多年的董事会经验，我们看到，**最不独立的董事当属那些依赖董事报酬生活的人**。更可笑的是，恰恰正是这些人被归类为"独立"董事。



### 四、独立董事的三个条件

> 💡 **最后底线**：如果你们赢，他们就大赢；如果你们输，他们就大输。这种方式或许可以被称之为**所有者资本主义**。

| 条件                        | 重要性                           | 说明                                                         |
| --------------------------- | -------------------------------- | ------------------------------------------------------------ |
| 1. **丰富的商业经验**       | 尤为难得，缺乏则其他两项作用有限 | 许多聪明、知名的社会人物对企业没有充足了解，不适合待在董事会内 |
| 2. **以股东利益为导向**     | 核心立场                         | 确保决策站在股东一边                                         |
| 3. **在公司拥有真正的利益** | 利益绑定                         | 与股东同甘共苦                                               |



### 五、伯克希尔的管理哲学

#### 5.1 两项核心工作

芒格和巴菲特只做两项工作：
1. **吸引和留住杰出的经理人**，管理不同类型的企业运营 
2. **资本配置**（见本书第2章和第6章）

#### 5.2 管理风格：让天才自由发挥

> 💡 **大卫·奥格威的哲学**：
> "如果我们雇用的都是比我们矮的人，我们将成为一家侏儒公司；但如果我们雇用的都是比我们高的人，我们会成为一家巨人公司。"

**伯克希尔的管理原则**：

| 原则           | 具体实践                                             |
| -------------- | ---------------------------------------------------- |
| 不干预运营     | 告诉杰出的CEO如何管理公司，简直愚蠢之极              |
| 消除烦冗仪式   | 经理人完全专注于自己制定的时间表                     |
| 模拟所有者心态 | 给每位经理人一个简单的任务，如同以下三种情况运营公司 |

**三种所有者心态模拟**：

1. 你拥有**100%**的股份
2. 将公司视为你和你的家族在世界上拥有的**唯一资产**，并且一直拥有
3. 至少在**100年**的时间里，不可以出售公司或与其他公司合并

> 💡 **推论**：不要让任何会计因素影响决策，哪怕是一丝一毫的影响。考虑什么是**重要的**，而不是考虑什么被**认为是重要的**。

#### 5.3 长期竞争优势："加宽护城河"

> 💡 **核心理念**：我们长期竞争优势的日渐提升，恰恰源自于这些日常不起眼的坚持，我们将其描述为**"加宽护城河"**。

**时间维度**：
- 大多数股票持有者打算持有至生命的终点
- 可以告诉CEO们着眼于**长期价值的最大化**，而不是下一个季度的盈利
- 永远不会以牺牲日益强大的竞争力为代价，取得短期的业绩表现

**日常经营原则**：
- ✅ 提升客户体验
- ✅ 削减不必要的成本
- ✅ 改进产品和服务
- ❌ 漠视客户
- ❌ 膨胀自大

> ⚠️ **警示**：如果管理层为了追逐短期的盈利而做出错误决策，会导致公司各方面处境恶化（成本、客户满意度、品牌美誉度），后续再做大量工作也无法弥补。看看如今汽车行业和航空行业的两难困境。

#### 5.4 用人哲学

> 💡 **核心信念**：我们只愿意与我们**喜欢并尊重**的人一起工作，这不仅仅将我们获得好结果的机会最大化，同时可以给我们带来非同一般的美好时光。

- 与大倒胃口的人一起工作就像为钱而结婚——**糟糕的主意**
- 大约**75%**的经理人已在财务上自由而独立
- 他们都是企业界的马克·麦克基韦斯（Mark McGwives）——**无须指点如何挥杆**

## C. 企业变化的焦虑

### 一、纺织业务的关闭与反思

#### 1.1 背景概述

1977年7月，伯克希尔决定关闭纺织业务。该业务的历史充满教训，以下是其关键历程：

- **1967年**：巴菲特合伙企业收购伯克希尔-哈撒韦，账面净资产为2200万美元，但内在价值远低于此。
- **前九年**：总销售额5.3亿美元，累计亏损1000万美元。
- **1967年初**：利用纺织业务产生的现金收购国民保险公司，进入保险业。

#### 1.2 保留纺织业务的初衷

在1978年的年报中，巴菲特列出了保留纺织业务的四个原因：

| 编号 | 原因说明                               |
| ---- | -------------------------------------- |
| 1    | 在当地是重要的就业雇主                 |
| 2    | 高管汇报问题直截了当，解决问题干劲十足 |
| 3    | 工人精诚合作，通情达理                 |
| 4    | 业务能产出适当回报 💡                   |

> ⚠️ **反思**：巴菲特后来承认，第4点判断错误。自1979年后，纺织业务持续大量消耗现金，直到1985年彻底关闭。

#### 1.3 为何没有更早关闭？

- 管理层肯·蔡斯和加里·莫里森非常出色，尝试了生产线更新、设备改造、分销布局优化。
- 收购沃姆贝克纺织厂试图产生协同效应，但最终失败。
- 巴菲特自省：“我忽略了哲学家孔德的忠告——聪明才智应是内心的仆人，而非奴隶。选择相信了我愿意相信的东西。”

#### 1.4 行业困境与投资陷阱

纺织行业处于全球化竞争红海中，国外工人工资远低于美国最低工资。尽管如此，伯克希尔的工人表现优异，工会也未提出不合理要求。

> 💡 **讽刺之处**：若工会行为不够友好，伯克希尔会更早意识到无望未来，从而提前关闭工厂，避免更大亏损。

#### 1.5 投资幻象：资本再投资的非理性循环

| 行为             | 结果                               |
| ---------------- | ---------------------------------- |
| 单个公司理性投资 | 降低成本，提高效率                 |
| 所有公司同时投资 | 成本下降被价格战抵消，整体回报不变 |
| 结论             | 总体呈现“非理性”状态               |

> 类比：就像人群中踮起脚尖看游行，所有人都更累了，但视野并未改善。

### 二、案例：伯灵顿工业公司

#### 2.1 数据对比表

| 指标            | 1964年   | 1985年               |
| --------------- | -------- | -------------------- |
| 销售额          | 12亿美元 | 28亿美元             |
| 股价            | 60美元   | 34美元（经拆股调整） |
| 再投资金额      | —        | 约30亿美元           |
| 每股再投资      | —        | 200美元              |
| 通胀倍数（CPI） | 1倍      | 3倍以上              |

> ⚠️ **结果**：尽管再投资巨大，销售额实际下降，利润率与净资产回报率远低于20年前，股东购买力缩水至1/3。

#### 2.2 启示

> 💡 **“你上了什么样的船，比你划船的效率更重要。”**

- 如果船长期漏水，换船比修补更有效。
- 管理才能虽重要，但无法抵挡行业基本面恶化。
- 正如巴菲特朋友所言：“如果你想得到一个好名声，一定要从事一个好行业。”

### 三、报业的兴衰与反思

#### 3.1 报业的黄金时代

- 20世纪大部分时间里，报纸是公众获取信息的主要途径。
- 广告商偏爱发行量最大的报纸，读者偏爱内容最丰富的报纸。
- **丛林法则**：大者生存，最终形成“一城一报”的垄断格局。

> 😄 一位出版商曾戏言：“我的财富归功于垄断和裙带关系。”

#### 3.2 变化的信号

早在1993年，巴菲特就警告：

> “媒体行业将被证明大不如前，不再像几年前我们想象的那样。”

但许多报业高管对自身行业的变化视而不见。

#### 3.3 当前挑战

| 因素                   | 影响                             |
| ---------------------- | -------------------------------- |
| 光纤、卫星电视、互联网 | 争夺眼球，报纸失去信息主导地位   |
| 免费、便捷的替代资源   | 互联网新闻盈利潜力远低于纸质时代 |
| 固定成本高             | 发行量下降加剧亏损风险           |

#### 3.4 伯克希尔的应对

- 《水牛城新闻报》在全美大型报纸中市场渗透率最高。
- 线上新闻也处于领先地位。
- 巴菲特表示：
  > “我们希望印刷版与线上版结合，探索可持续的商业模式。”

#### 3.5 为何仍保留报纸？

- 对当地居民而言，报纸就像运动队，具有荣誉感和影响力。
- 有钱人愿意购买报纸，服务社区。
- 巴菲特和芒格每天读五份报纸，热爱新闻业。

> ⚠️ 但必须谨慎：没有规定保证收入不会低于成本。报纸的“超级优越感”会随着重要性消失而褪色。

### 四、总结与启示

| 主题     | 核心启示                                 |
| -------- | ---------------------------------------- |
| 纺织业务 | 不要因为历史或情感而持续投资于无望的行业 |
| 资本配置 | 理性个体行为可能导致群体非理性结果       |
| 报业     | 行业基本面恶化时，管理才能只能减缓衰退   |
| 投资哲学 | 选择正确的“船”，比划船技巧更重要         |

> 💡 **巴菲特名言**：“英雄还要靠时势，管理技能的杰出依然敌不过大环境基本面的恶化。

## D. 社会契约

### 一、核心特征对比

以下为 BNSF 铁路公司与中美能源公司的共同特征，区别于伯克希尔体系内其他企业：

| 特征维度     | 具体说明                                                     |
| :----------- | :----------------------------------------------------------- |
| **资产结构** | 拥有大量由**长期负债**支持的受管制长期资产（伯克希尔不提供担保） |
| **盈利能力** | 自身盈利足以覆盖利息，即使面临极端不利的经济环境             |
| **监管环境** | 受政府严格管制，需持续进行工厂与设备的大规模投资             |
| **经营要求** | 提供高效、令客户满意的服务，赢得当地民众与管理当局的尊重     |
| **核心回报** | 被允许获得**合理的资本投资回报**                             |



### 二、BNSF 铁路公司：美国经济的流通命脉

#### 2.1 市场份额与重要性

- 铁路承担了全美 **42%** 的城际间货运量（以“吨位-英里”衡量）
- BNSF 运量占全行业 **28%**，换算后提供全美超过 **11%** 的城际间吨位-英里运输量
- 随着西部人口流动，市场份额预计进一步增长

#### 2.2 社会责任与资本投入

> 我们是美国经济流通系统的主要和重要部分，有责任不断维护并改善 23,000 英里铁路及其辅助设施。

- **2011 年维护与改进支出**：超过 **20 亿美元**（定期超出折旧额）
- **投资原则**：预见社会需求，而非被动反馈
- **回报预期**：在智慧监管与智慧投资的协同下获得合理回报 💡



### 三、中美能源公司：公用事业的社会契约典范

#### 3.1 业务覆盖规模

| 业务领域         | 覆盖范围 / 数据                                              |
| :--------------- | :----------------------------------------------------------- |
| **电力供应**     | 向 **240 万** 美国客户供电；爱荷华州、怀俄明州、犹他州最大电力供应商 |
| **天然气管道**   | 提供全美 **8%** 的天然气输送                                 |
| **伯克希尔持股** | 持有 **89.8%** 股份                                          |

#### 3.2 管道业务转型成果

- **北方天然气公司**：2002 年被收购时，行业权威评级为 **倒数第一**（43 家公司中排第 43 名）
- **最新评级**：跃升至 **第 2 名** 🏆
- **第 1 名**：同为旗下克恩河公司

#### 3.3 电力价格与风电投资

| 项目                      | 数据 / 表现                                               |
| :------------------------ | :-------------------------------------------------------- |
| **爱荷华州电价**          | 自 1999 年收购以来 **从未涨价**                           |
| **同行对比**              | 其他大型电力公司同期提价 **超过 70%**，电价远高于中美能源 |
| **风电装机（2011 年底）** | **2,909 兆瓦** – 超过美国任何其他受管制电厂               |
| **风电投资总额**          | **54 亿美元**（已投入或承诺）                             |
| **盈利留存**              | 全部盈利保留用于再投资，不同于大多数公用事业公司分红      |

#### 3.4 监管与社会契约

- 社会以**低价供应**回馈公司，监管者几乎**毫无例外地允许新增投资资本获得合理回报**
- 未来承诺：以当地希望的方式服务，并相信能获得应得的投资回报 🤲



### 四、共同的社会契约原则

> 无论是铁路还是公用事业，都存在一种**社会契约**：若任何一方未能履行其职责，双方都将不可避免地遭受损失。

- **契约双方**应主动采取鼓励对方良好行为的方式
- 没有第一流的电力和铁路系统，美国无法实现其巨大的经济潜力
- 伯克希尔承诺：**为实现这一目标扮演好自己的角色，做出应有的贡献**

## E. 由股东决策的公司捐赠方法

### 一、核心原则：股东主导 vs. 管理层主导

巴菲特认为，公司的慈善行为应当反映**股东的偏好**，而非高管或董事的个人意愿。他尖锐地指出：

> 💡 “如果A拿着B的钱给C，而A又是立法者，这叫做纳税。但当A是公司高管或董事时，这叫作慈善。”

#### 1.1 两类公司捐赠

| 类型         | 定义                                 | 伯克希尔的做法             |
| ------------ | ------------------------------------ | -------------------------- |
| **直接受益** | 捐赠带来与金额大体相当的公司直接利益 | ✅ 参与，但总额保持较低水平 |
| **间接受益** | 利益难以衡量、反馈滞后               | ❌ 几乎从不参与             |

#### 1.2 与其他公司的对比

| 特征         | 伯克希尔             | 多数上市公司       |
| ------------ | -------------------- | ------------------ |
| 捐赠决策主体 | 股东（按持股比例）   | 管理层或董事会     |
| 决策依据     | 股东个人慈善偏好     | 同行压力、社会压力 |
| 税务效率     | 股东可免除个人所得税 | 无此安排           |
| 公司治理理念 | **合伙制心态**       | 管理层代行所有权   |



### 二、税务裁定与运作机制

1981年9月30日，伯克希尔收到美国财政部的关键税务裁定：

- 📄 **机制**：每位股东按持股比例，可指定慈善受赠机构 → 伯克希尔代写支票
- 💰 **税务优惠**：股东无需为该项捐赠缴纳个人所得税
- 👥 **参与资格**：在股东名册中有真实姓名记录的股东

> ⚠️ **注意**：该计划适用于公司层面有足够税务抵扣额度的年份（大约每十年中有两三年无法进行）。

#### 2.1 运作流程

1. 每年10月10日左右，通知股东每股可指定捐赠的额度
2. 附上回复表格
3. 股东有三周时间回复



### 三、计划成效与股东参与度

#### 3.1 股东稳定性（与“旋转门”机构形成对比）

- 📊 每年末，**超过98%** 的股票仍由年初的股东持有
- 体现真正的“所有者心态”

#### 3.2 参与率（创纪录）

| 指标                       | 数据       |
| -------------------------- | ---------- |
| 有资格参与的股份数         | 932,206 股 |
| 回应率（全体）             | **95.6%**  |
| 剔除巴菲特相关股份后回应率 | **90%**    |
| 自发写信赞成的股东         | > 3%       |

> 💡 此结果是在**不提供回邮信封**的情况下达成的，凸显了股东的高度认同。

#### 3.3 累计贡献（1980–2002）

| 指标             | 数据                                 |
| ---------------- | ------------------------------------ |
| 累计捐赠总额     | **1.97亿美元**                       |
| 年度捐赠额增长   | 从1–2百万美元 → 1700万美元（2002年） |
| 受赠机构数量增长 | 从<1000个 → 3500个                   |

### 五、争议与终止（2003年）

#### 5.1 导火索：堕胎争议引发的抵制

- 关于堕胎合法化的对立双方持续向伯克希尔施压
- 部分反对者**联合抵制**伯克希尔旗下产品
- 受影响最严重的是 **宠厨（Pampered Chef）** 的独立加盟商
  - 这些加盟商既非员工，也未参与公司决策
  - 因抵制行为遭受**严重收入损失**

#### 5.2 终止决策的逻辑

> ⚖️ “为了我们和其他股东得到一些可怜的税务效应，而伤害那些正直善良、努力工作的人们，实在没有仁慈可言。” —— 巴菲特

**决策结果**：
- ❌ 母公司层面不再做任何捐赠
- ✅ 各子公司延续原有慈善政策，但**个人化捐赠须由管理者自掏腰包**
