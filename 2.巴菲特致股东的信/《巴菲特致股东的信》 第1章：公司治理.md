## A. 完整公平的信息披露

> **核心原则**：在伯克希尔公司，我们通常会**换位思考**，设想如果我们自己处于股东的位置，应该得到什么样的信息，我们会从这样的角度向股东披露完整的报告。

### 一、信息披露的核心理念

芒格和我会公开当前状况下，公司的**所有重要的运营状况**，以及公司高管对于**长期商业经济特征的坦诚看法**。无论是很多财务细节，还是任何重大的运营数据的讨论，我们都希望给出**解读**。

- **我们反对的内容**：
  - ❌ 人物、工厂、产品的照片
  - ❌ 引用 **EBITDA**（未计利息、税项、折旧及摊销前的利润）的概念
  - ❌ 模糊不清的会计技巧
  - ❌ 由公司公关部或财经公关顾问发出的信息

- **我们期待的内容**：
  - ✅ 公司CEO能**直接用自己的语言**解释所发生的事情
  - ✅ 坦诚、清晰、无修饰的运营状况说明

> 💡 **技巧**：巴菲特和芒格认为，引用EBITDA的概念会让他们"不寒而栗"——难道管理层认为公司的资本开支会由牙齿仙女（Tooth Fairy）承担吗？



### 二、公平报告的定义

对于我们而言，所谓**"公平报告"**意味着：

| 要素         | 具体做法                                                     |
| ------------ | ------------------------------------------------------------ |
| **平等获取** | 我们的30万个"合伙人"（指伯克希尔的股东）能够在**同一时间**或尽可能同时得到信息 |
| **发布时间** | 选择在**周五股市收盘后**，或**次日的早晨**，将年度和季度财报发布在互联网上 |
| **消化时间** | 让股东或有兴趣的潜在投资者有充分的时间，在**周一开盘之前**消化信息 |



### 三、反对选择性披露

⚠️ **注意**：我们要为美国证券交易委员会（SEC）主席**小阿瑟·莱维特（Arthur Levitt, Jr.）**进行的股市打假活动鼓掌。

| 问题           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| **选择性披露** | 像癌症一样蔓延，已成为大型公司的标准化行为                   |
| **操作方式**   | 用于"指引"分析师或大股东对于盈利的预期，提供的数据会正好或略低于公司真实预期 |
| **本质**       | 这就像抛媚眼或点头一样，给那些满脑子投机思维的机构和顾问发出暗示 |
| **危害**       | 相对于以投资为导向的个人投资者，这些人得到了更多的信息优势   |
| **性质**       | 这实际上是一种**贪污腐败行为**，但却不幸地被华尔街和美国企业界奉为圭臬 |

> 感谢莱维特主席代表投资人利益而进行的不屈不挠且卓有成效的努力。现在，所有公司都被要求平等地对待他们的股东。但这种变革，不是出于道德良心，而是出于强迫压力，这对于公司CEO们以及公司的投资者关系部门而言，并不是一件光彩的事。



### 四、CEO盈利预测的警示

芒格和我们都认为，**CEO们预测公司增长速度，是骗人的和危险的**。当然，他们是被分析师和公司自己的投资者关系部门怂恿裹挟着这么做。但他们应该拒绝，因为这样预测多了会招致麻烦。

**为什么盈利预测是危险的？**

1. **不切实际的增长预期**
   
   - 只有非常少的大型公司能维持15%的年复合增长率
   - 测试：假设从1970年到1980年选出200家盈利最高的公司，看看有多少家可以保持15%的每股盈利增长率——**很少有公司可以满足这个标准**
   - 巴菲特打赌：2000年最赚钱的200家公司中，能在未来20年中保持15%每股盈利增长的公司，将**不超过10家**
   
2. **腐蚀CEO行为**
   - 许多CEO们参与那些并不经济的企业运作，仅仅是为了追求他们曾经宣布的盈利目标
   - 在使出重重解数、筋疲力尽之后，他们有时甚至玩起了各种各样的会计游戏，以"制造数字"，达成目标

3. **滚雪球效应**
   
   - 一旦公司将盈利从一个时期转移到另一个时期，运营的缺口将会随之在未来要求更多的会计操纵
   - 更加肆无忌惮，愚蠢就变成了欺诈
   - > "据悉，更多的钱是被'笔'偷走的，而不是被'枪'劫走的。"



### 五、给投资者的三个建议

以下是将“给投资者的三个建议”内容整理成的表格形式，便于快速对照和记忆：

| 建议编号 | 建议主题                     | 警示信号                                 | 具体说明 / 补充内容                                          |
| :------: | :--------------------------- | :--------------------------------------- | :----------------------------------------------------------- |
|    1     | 警惕“弱会计”的公司           | 公司没有费用选项，或养老金假设如天马行空 | 管理层在公开方面使用低下的手段，背后也可能如此。<br>➤ 厨房里如果有蟑螂，绝不可能只有一只。 |
|    2     | 警惕“不知所云的注脚”         | 看不懂财务报告中的注释或管理解释         | CEO可能不想让你明白。安然公司的某些交易描述至今令巴菲特困惑。<br><br>**关于EBITDA的误导性：**<br>鼓吹EBITDA有害，认为折旧“非现金”所以不是真实费用是错的。折旧在资产产生收益前就已支出现金。<br>➤ 比喻：公司一次性预付10年工资，后9年记录为“非现金”成本，但这并不代表真实成本消失。 |
|    3     | 警惕“大肆鼓吹的盈利成长预测” | 企业很少能在毫无意外的环境中顺利发展     | 所有盈利都不是简单顺利可预知的。                             |



### 六、伯克希尔的坦诚

芒格和我**不仅不知道伯克希尔下一年的盈利水平，甚至不知道下一个季度的盈利如何**。

> 对于那些一贯声称能够预知未来的CEO们，我们抱怀疑态度。如果他们总是能达到他们预测的目标，我们对其会是**彻头彻尾的怀疑**。那些习惯于"制造数字"的公司高管们，在某种程度上是试图粉饰数字。



## B. 董事会与公司高管

### 一、CEO与基层员工的考核差异

#### 1.1 基层岗位：清晰可量化的标准

基层职位通常具有**清晰的标准**和**可量化的表现指标**，不达标即出局：

| 岗位类型 | 考核标准                | 不达标后果   |
| -------- | ----------------------- | ------------ |
| 秘书     | 打字速度（如80字/分钟） | 立即失去工作 |
| 销售人员 | 销售目标                | 立刻被解聘   |

#### 1.2 CEO岗位：标准缺失与监督缺位

与基层员工形成鲜明对比，**CEO的考核存在系统性缺陷**：

- ⚠️ **衡量标准几乎不存在**：碌碌无为的CEO经常可以蒙混过关
- ⚠️ **缺乏即刻的监督考察机制**：即使公司运营败象毕露，CEO仍可推诿塞责
- **"先射箭后画靶"现象**：匆忙将落箭之点圈为靶心，事后合理化失败

#### 1.3 职位越高，跌得越轻

| 层级           | 监督强度 | 问责后果           | 利益保障                         |
| -------------- | -------- | ------------------ | -------------------------------- |
| 基层员工       | 高       | 立即解雇           | 无                               |
| 中层管理者     | 中       | 影响个人利益       | 有限                             |
| CEO/董事会成员 | 低       | 含混其辞、推诿塞责 | 即使公司被收购接管，仍获巨额利益 |

> ⚠️ **讽刺现象**：在董事会议上，对CEO成绩表现的批评就像打嗝一样无关痛痒，而不称职的打字员却无法享受这种待遇。



### 二、董事会里的三种情形

巴菲特指出，上市公司中存在三种根本不同的治理情形，尽管董事们的法律责任完全相同：

#### 2.1 三种情形对比

| 情形  | 股权结构特征           | 典型案例         | 董事会角色                             | 有效性                                             |
| ----- | ---------------------- | ---------------- | -------------------------------------- | -------------------------------------------------- |
| **1** | 无控股股东             | 大多数上市公司   | 应像缺席的股东一样，着眼于公司长期利益 | ⚠️ 容易被"长期"一词滥用，董事缺乏诚信或独立思考能力 |
| **2** | 控股股东同时是公司高管 | 伯克希尔公司     | 无须扮演中间人角色，除反对外别无他法   | 取决于控股股东品质                                 |
| **3** | 控股股东不参与公司管理 | 好时食品、道琼斯 | 外部董事可向大股东直接反映不满         | ✅ **逻辑上最为有效**                               |

#### 2.2 第一种情形的治理建议

> 💡 **理想董事会构成**：
> - 成员人数：**十个或更少**
> - 多数应来自**公司外部**
> - 外部董事应**制定CEO业绩衡量标准**
> - 定期召开**没有CEO参与**的会议，以衡量标准评判其业绩

**董事会成员必备特质**：
1. 精通业务
2. 乐于本职
3. 具有**股东利益导向**精神

> ⚠️ **错误做法**：基于社会名声或多元化因素挑选董事。和和气气但无所事事的董事一旦被任命，再无丢失工作之虞。

#### 2.3 第二种情形下的外部董事选择

如果股东/公司高管是平庸或糟糕之辈，董事除反对外无能为力。

> ⚠️ **关键行动**：如果情况没有改观，问题十分严重，**外部董事应该辞职**。他们的辞职将是对管理质疑的警示信号，强调已没有外部人士可以更正股东或公司高管的缺点。

#### 2.4 第三种情形的优势

- 大股东既不用自我批评，也不用麻烦地面临争取多数董事同意
- 可保证挑选的外部董事能给董事会带来积极有益的因素
- 外部董事知道其建议能被认真倾听
- 若大股东聪明且自信，会做出既尊重管理层又维护股东利益的英明决策



### 三、"独立"董事的真相

#### 3.1 当前"独立"董事的呼声与实质

> 💡 **巴菲特的支持**：由外部董事定期召开没有CEO参与的董事会，这种形式是一种改革，我对此持**热烈支持态度**。

**真正的"独立"董事应具备**：
- 能思考独立、观点独立
- **必须精通商务**
- 有兴趣并具有**股东利益导向**的特质

#### 3.2 "独立"董事的失败案例：投资公司董事会

**历时62年的案例研究（1942年以来）**：

| 职责         | 应然表现     | 实然表现                                     |
| ------------ | ------------ | -------------------------------------------- |
| 选择投资经理 | 寻找最佳人选 | 僵化刻板：A基金永远选A经理，B基金永远选B经理 |
| 谈判管理费用 | 争取最低费用 | 董事们毫不关心，降费对基金经理影响重大       |

> ⚠️ **讽刺对比**：当董事们处理自己的金钱时，他们会另辟蹊径、谨慎小心；肩负其他投资人托付时，心中却不这么想。

#### 3.3 并购案中的虚伪体现

在A公司被高价卖给B经理的案例中：
- "独立"董事们开始自我反省，认为B经理才是最佳人选
- B经理买下A公司后，身价即刻飞涨（因需补偿收购成本，从A公司股东口袋里捞回）
- 参考：约翰·博格《公募基金的常识》

#### 3.4 伯克希尔董事会的典范标准

以《圣经》"你的财富在哪里，你的心就在哪里"为标准：

| 标准     | 伯克希尔实践                                      |
| -------- | ------------------------------------------------- |
| 持股要求 | 每位董事至少将**400万美元以上**的身家放在伯克希尔 |
| 持股来源 | 没有任何股份是靠股票期权或赠予获得                |
| 酬劳水平 | 相较于自身年收入极其有限                          |
| 风险承担 | 没有替董事安排任何责任保险                        |

> 💡 **核心结论**：芒格和我已看过很多符合《圣经》"财富"观点的行为。根据多年的董事会经验，我们看到，**最不独立的董事当属那些依赖董事报酬生活的人**。更可笑的是，恰恰正是这些人被归类为"独立"董事。



### 四、独立董事的三个条件

> 💡 **最后底线**：如果你们赢，他们就大赢；如果你们输，他们就大输。这种方式或许可以被称之为**所有者资本主义**。

| 条件                        | 重要性                           | 说明                                                         |
| --------------------------- | -------------------------------- | ------------------------------------------------------------ |
| 1. **丰富的商业经验**       | 尤为难得，缺乏则其他两项作用有限 | 许多聪明、知名的社会人物对企业没有充足了解，不适合待在董事会内 |
| 2. **以股东利益为导向**     | 核心立场                         | 确保决策站在股东一边                                         |
| 3. **在公司拥有真正的利益** | 利益绑定                         | 与股东同甘共苦                                               |



### 五、伯克希尔的管理哲学

#### 5.1 两项核心工作

芒格和巴菲特只做两项工作：
1. **吸引和留住杰出的经理人**，管理不同类型的企业运营 
2. **资本配置**（见本书第2章和第6章）

#### 5.2 管理风格：让天才自由发挥

> 💡 **大卫·奥格威的哲学**：
> "如果我们雇用的都是比我们矮的人，我们将成为一家侏儒公司；但如果我们雇用的都是比我们高的人，我们会成为一家巨人公司。"

**伯克希尔的管理原则**：

| 原则           | 具体实践                                             |
| -------------- | ---------------------------------------------------- |
| 不干预运营     | 告诉杰出的CEO如何管理公司，简直愚蠢之极              |
| 消除烦冗仪式   | 经理人完全专注于自己制定的时间表                     |
| 模拟所有者心态 | 给每位经理人一个简单的任务，如同以下三种情况运营公司 |

**三种所有者心态模拟**：

1. 你拥有**100%**的股份
2. 将公司视为你和你的家族在世界上拥有的**唯一资产**，并且一直拥有
3. 至少在**100年**的时间里，不可以出售公司或与其他公司合并

> 💡 **推论**：不要让任何会计因素影响决策，哪怕是一丝一毫的影响。考虑什么是**重要的**，而不是考虑什么被**认为是重要的**。

#### 5.3 长期竞争优势："加宽护城河"

> 💡 **核心理念**：我们长期竞争优势的日渐提升，恰恰源自于这些日常不起眼的坚持，我们将其描述为**"加宽护城河"**。

**时间维度**：
- 大多数股票持有者打算持有至生命的终点
- 可以告诉CEO们着眼于**长期价值的最大化**，而不是下一个季度的盈利
- 永远不会以牺牲日益强大的竞争力为代价，取得短期的业绩表现

**日常经营原则**：
- ✅ 提升客户体验
- ✅ 削减不必要的成本
- ✅ 改进产品和服务
- ❌ 漠视客户
- ❌ 膨胀自大

> ⚠️ **警示**：如果管理层为了追逐短期的盈利而做出错误决策，会导致公司各方面处境恶化（成本、客户满意度、品牌美誉度），后续再做大量工作也无法弥补。看看如今汽车行业和航空行业的两难困境。

#### 5.4 用人哲学

> 💡 **核心信念**：我们只愿意与我们**喜欢并尊重**的人一起工作，这不仅仅将我们获得好结果的机会最大化，同时可以给我们带来非同一般的美好时光。

- 与大倒胃口的人一起工作就像为钱而结婚——**糟糕的主意**
- 大约**75%**的经理人已在财务上自由而独立
- 他们都是企业界的马克·麦克基韦斯（Mark McGwives）——**无须指点如何挥杆**



## C. 企业变化的焦虑

### 一、纺织业务的关闭与反思

#### 1.1 背景概述

1977年7月，伯克希尔决定关闭纺织业务。该业务的历史充满教训，以下是其关键历程：

- **1967年**：巴菲特合伙企业收购伯克希尔-哈撒韦，账面净资产为2200万美元，但内在价值远低于此。
- **前九年**：总销售额5.3亿美元，累计亏损1000万美元。
- **1967年初**：利用纺织业务产生的现金收购国民保险公司，进入保险业。

#### 1.2 保留纺织业务的初衷

在1978年的年报中，巴菲特列出了保留纺织业务的四个原因：

| 编号 | 原因说明                               |
| ---- | -------------------------------------- |
| 1    | 在当地是重要的就业雇主                 |
| 2    | 高管汇报问题直截了当，解决问题干劲十足 |
| 3    | 工人精诚合作，通情达理                 |
| 4    | 业务能产出适当回报 💡                   |

> ⚠️ **反思**：巴菲特后来承认，第4点判断错误。自1979年后，纺织业务持续大量消耗现金，直到1985年彻底关闭。

#### 1.3 为何没有更早关闭？

- 管理层肯·蔡斯和加里·莫里森非常出色，尝试了生产线更新、设备改造、分销布局优化。
- 收购沃姆贝克纺织厂试图产生协同效应，但最终失败。
- 巴菲特自省：“我忽略了哲学家孔德的忠告——聪明才智应是内心的仆人，而非奴隶。选择相信了我愿意相信的东西。”

#### 1.4 行业困境与投资陷阱

纺织行业处于全球化竞争红海中，国外工人工资远低于美国最低工资。尽管如此，伯克希尔的工人表现优异，工会也未提出不合理要求。

> 💡 **讽刺之处**：若工会行为不够友好，伯克希尔会更早意识到无望未来，从而提前关闭工厂，避免更大亏损。

#### 1.5 投资幻象：资本再投资的非理性循环

| 行为             | 结果                               |
| ---------------- | ---------------------------------- |
| 单个公司理性投资 | 降低成本，提高效率                 |
| 所有公司同时投资 | 成本下降被价格战抵消，整体回报不变 |
| 结论             | 总体呈现“非理性”状态               |

> 类比：就像人群中踮起脚尖看游行，所有人都更累了，但视野并未改善。

### 二、案例：伯灵顿工业公司

#### 2.1 数据对比表

| 指标            | 1964年   | 1985年               |
| --------------- | -------- | -------------------- |
| 销售额          | 12亿美元 | 28亿美元             |
| 股价            | 60美元   | 34美元（经拆股调整） |

> ⚠️ **结果**：尽管再投资巨大，销售额实际下降，利润率与净资产回报率远低于20年前，股东购买力缩水至1/3。

#### 2.2 启示

> 💡 **“你上了什么样的船，比你划船的效率更重要。”**

- 如果船长期漏水，换船比修补更有效。
- 管理才能虽重要，但无法抵挡行业基本面恶化。
- 正如巴菲特朋友所言：“如果你想得到一个好名声，一定要从事一个好行业。”

### 三、报业的兴衰与反思

#### 3.1 报业的黄金时代

- 20世纪大部分时间里，报纸是公众获取信息的主要途径。
- 广告商偏爱发行量最大的报纸，读者偏爱内容最丰富的报纸。
- **丛林法则**：大者生存，最终形成“一城一报”的垄断格局。

> 😄 一位出版商曾戏言：“我的财富归功于垄断和裙带关系。”

#### 3.2 变化的信号

早在1993年，巴菲特就警告：

> “媒体行业将被证明大不如前，不再像几年前我们想象的那样。”

但许多报业高管对自身行业的变化视而不见。

#### 3.3 当前挑战

| 因素                   | 影响                             |
| ---------------------- | -------------------------------- |
| 光纤、卫星电视、互联网 | 争夺眼球，报纸失去信息主导地位   |
| 免费、便捷的替代资源   | 互联网新闻盈利潜力远低于纸质时代 |
| 固定成本高             | 发行量下降加剧亏损风险           |

#### 3.4 伯克希尔的应对

- 《水牛城新闻报》在全美大型报纸中市场渗透率最高。
- 线上新闻也处于领先地位。
- 巴菲特表示：
  > “我们希望印刷版与线上版结合，探索可持续的商业模式。”

#### 3.5 为何仍保留报纸？

- 对当地居民而言，报纸就像运动队，具有荣誉感和影响力。
- 有钱人愿意购买报纸，服务社区。
- 巴菲特和芒格每天读五份报纸，热爱新闻业。

> ⚠️ 但必须谨慎：没有规定保证收入不会低于成本。报纸的“超级优越感”会随着重要性消失而褪色。

| 资本配置 | 理性个体行为可能导致群体非理性结果       |

## D. 社会契约

### 一、核心特征对比

以下为 BNSF 铁路公司与中美能源公司的共同特征，区别于伯克希尔体系内其他企业：

| 特征维度     | 具体说明                                                     |
| :----------- | :----------------------------------------------------------- |
| **资产结构** | 拥有大量由**长期负债**支持的受管制长期资产（伯克希尔不提供担保） |
| **盈利能力** | 自身盈利足以覆盖利息，即使面临极端不利的经济环境             |
| **监管环境** | 受政府严格管制，需持续进行工厂与设备的大规模投资             |
| **经营要求** | 提供高效、令客户满意的服务，赢得当地民众与管理当局的尊重     |
| **核心回报** | 被允许获得**合理的资本投资回报**                             |



### 二、BNSF 铁路公司：美国经济的流通命脉

#### 2.1 市场份额与重要性

- 随着西部人口流动，市场份额预计进一步增长

#### 2.2 社会责任与资本投入

> 我们是美国经济流通系统的主要和重要部分，有责任不断维护并改善 23,000 英里铁路及其辅助设施。

- **2011 年维护与改进支出**：超过 **20 亿美元**（定期超出折旧额）
- **投资原则**：预见社会需求，而非被动反馈
- **回报预期**：在智慧监管与智慧投资的协同下获得合理回报 💡

- 没有第一流的电力和铁路系统，美国无法实现其巨大的经济潜力
- 伯克希尔承诺：**为实现这一目标扮演好自己的角色，做出应有的贡献** 



## E. 由股东决策的公司捐赠方法

### 一、核心原则：股东主导 vs. 管理层主导

巴菲特认为，公司的慈善行为应当反映**股东的偏好**，而非高管或董事的个人意愿。他尖锐地指出：

> 💡 “如果A拿着B的钱给C，而A又是立法者，这叫做纳税。但当A是公司高管或董事时，这叫作慈善。”

#### 1.1 两类公司捐赠

| 类型         | 定义                                 | 伯克希尔的做法             |
| ------------ | ------------------------------------ | -------------------------- |
| **直接受益** | 捐赠带来与金额大体相当的公司直接利益 | ✅ 参与，但总额保持较低水平 |
| **间接受益** | 利益难以衡量、反馈滞后               | ❌ 几乎从不参与             |

#### 1.2 与其他公司的对比

| 特征         | 伯克希尔             | 多数上市公司       |
| ------------ | -------------------- | ------------------ |
| 捐赠决策主体 | 股东（按持股比例）   | 管理层或董事会     |
| 决策依据     | 股东个人慈善偏好     | 同行压力、社会压力 |
| 税务效率     | 股东可免除个人所得税 | 无此安排           |
| 公司治理理念 | **合伙制心态**       | 管理层代行所有权   |



### 二、税务裁定与运作机制

1981年9月30日，伯克希尔收到美国财政部的关键税务裁定：

- 📄 **机制**：每位股东按持股比例，可指定慈善受赠机构 → 伯克希尔代写支票
- 💰 **税务优惠**：股东无需为该项捐赠缴纳个人所得税
- 👥 **参与资格**：在股东名册中有真实姓名记录的股东

> ⚠️ **注意**：该计划适用于公司层面有足够税务抵扣额度的年份（大约每十年中有两三年无法进行）。

#### 2.1 运作流程

1. 每年10月10日左右，通知股东每股可指定捐赠的额度
2. 附上回复表格
3. 股东有三周时间回复



### 三、计划成效与股东参与度

#### 3.1 股东稳定性（与“旋转门”机构形成对比）

- 📊 每年末，**超过98%** 的股票仍由年初的股东持有
- 体现真正的“所有者心态”

#### 3.2 参与率（创纪录）

| -------------------------- | ---------- |

| 剔除巴菲特相关股份后回应率 | **90%**    |
| 自发写信赞成的股东         | > 3%       |

> 💡 此结果是在**不提供回邮信封**的情况下达成的，凸显了股东的高度认同。

#### 3.3 累计贡献（1980–2002）

## F. 公司高管的报酬原则

### 一、留存利润增长 ≠ 管理成就

#### 1.1 储蓄账户的复利魔术

当资本回报率平淡时，“投得越多、赚得越多”的记录并不值得称赞。将储蓄账户本金翻四倍，利息自然同比例增长，没人会因此夸奖你。然而，许多 CEO 退休时却因在位期间将公司盈利提升四倍而备受赞誉，却未审视这种增长是否仅仅来自留存利润的复利积累。

用简单的储蓄账户演示：
- 本金：10万美元
- 利率：8%
- 派息率：25%（剩余75%作为“留存盈利”滚存）
- 受托管理人有十年期固定行权价的期权

十年后结果：

| 项目     | 起始（第1年） | 第十年末 |
| -------- | ------------- | -------- |
| 账户价值 | $100,000      | $179,084 |
| 年度盈利 | $8,000        | $13,515  |
| 年度分红 | $2,000        | $3,378   |

📊 管理人的报告会充斥着直刺天穹的图表——但这一切增长仅仅源于留存利润的复利，而非高效运用资金。

💡 **关键洞察：盈利增加可能并非管理层杰出。**



### 二、股票期权的双重标准与内在不公

#### 2.1 对外苛刻，对内慷慨

企业世界里，没有什么值得将十年期固定行权价期权授予外人——这是普遍共识。但管理层在授予自己同样期权时，态度截然不同。

> “与自己谈判很少会引发酒吧里的斗殴。”

那些为自己设计十年期固定价格期权的管理层：
- 漠视留存利润自行增值的事实
- 漠视资本的成本
- 所获收益就像坐在储蓄账户上自动增值一样

⚠️ 讽刺案例：克莱斯勒在政府救助中授予政府股票期权，当政府获利后，公司却抱怨政府所得与贡献“不相称”。然而，高管们从未因自己获得超额期权而感到苦恼。

#### 2.2 期权持有人与股东：两条完全不同的船

期权常被美化成“将管理层与股东放在同一条船上”，实则不然：

| 维度     | 股东                 | 固定价格期权持有人           |
| -------- | -------------------- | ---------------------------- |
| 资本成本 | 承担全部资本成本     | **无资本成本负担**           |
| 下行风险 | 参与全部下跌风险     | **无下跌风险**               |
| 分红偏好 | 希望合理分红         | 不分红更有利，可多分留存利润 |
| 利益对齐 | 追求公司真实价值成长 | 追求股价波动与留存复利       |

在分红政策上同样冲突：期权持有人倾向于低分红甚至零分红，以最大化留存利润带来的期权增值，而这可能伤害股东利益。



### 三、巴菲特对期权的三点核心看法

#### 3.1 应与公司整体表现挂钩

期权应授予负整体责任的管理层。局部负责人应依其所负责部分的表现获得报酬——一个0.350的优秀击球手值得嘉奖，即便球队整体不佳；反之亦然。

#### 3.2 结构应精心设计

期权设计必须考虑：
- 公司**留存利润**的自然增长
- 资本附带**成本**
- 依据**真实价值**定价

奇怪的是，当面对外部收购报价时，管理层能精准指出报价如何低估公司真正价值；但在授予自己期权时，却欣然接受同样的压价。

#### 3.3 尊重优秀文化，但避免滥用

一些极其敬佩的经理人通过领导力建立了以期权为有效工具的公司文化。这种文化极为少见，应当保留。就像俗话说的：

> “直到用破了再修理”总好过“为了纯洁不惜任何代价”。  
> ——（译者注：相当于中文语境中“水至清则无鱼”）



### 四、期权如何侵蚀公司价值：一个量化案例

以 **停滞公司 CEO 弗莱德·徒劳** 为例：
- 净资产：$100亿，年盈利：$10亿
- 总股本：1亿股，每股盈利（EPS）：$10
- 市盈率（PE）：10倍，股价：$100
- CEO 获 1% 股权的十年期固定行权价期权
- 十年内**全部盈利用于回购股份，不派息**

#### 十年后变化

| 指标          | 期初  | 第十年末    | 变化   |
| ------------- | ----- | ----------- | ------ |
| 总股本        | 1亿股 | 3,870万股   | -61.3% |
| 每股盈利      | $10   | $25.80      | +158%  |
| 股价（PE=10） | $100  | $258        | +158%  |
| CEO期权价值   | —     | **$1.58亿** | —      |

公司业务毫无增长，仅凭留存利润回购，弗莱德便暴富。

#### 更惊人的情景

- 若十年间**利润下滑20%**，弗莱德仍可获利**超过$1亿**。
- 若用留存利润投资于回报率仅**5%** 的差劲项目，只要PE维持10倍，期权仍值 **$630万**。

💸 股东不禁要问：说好的“利益联盟”在哪里？正常的派息政策（如1/3分红）虽能减缓极端结果，但仍会给毫无作为的管理层丰厚回报。

CEO们深知：**每派发一分钱股息，就会降低他们手中期权的价值。** 但在代理文件中，他们从不提及这种利益冲突——尽管他们总在内部说“资本有成本”，授予自己期权时，资本却是免费的。

> 💡 巴菲特连用两次“令人吃惊”：这种随留存利润自动调整行权价的期权，几十年从未被发行过。报酬设计“专家”对此陌生，但设计对管理层有利的方案时却是百科全书——“吃谁的面包，唱谁的歌。”



### 五、伯克希尔的激励哲学：业绩挂钩与资本有成本

#### 5.1 奖励与可控职责挂钩

伯克希尔奖励的是在其职责范围内的经营目标达成：
- 喜诗糖果干得好，不奖励新闻公司；反之亦然。
- 奖金支票与伯克希尔股价表现**无关**。
- 优秀表现应得奖励，即便股价下跌；平庸表现不应得特别奖励，即便股价飙升。
- “工作表现”的定义不视行业经济状况而定：顺风与逆风应区别看待。

激励可非常巨大：顶尖经理人奖金可达基本工资的**五倍或更多**，上不封顶，不分等级。20岁与40岁的0.300击球手价值相同。

✅ **经理人用奖金（甚至借钱）在公开市场购买伯克希尔股票，真正与股东穿同一双鞋。**

#### 5.2 资本不免费：双向对称机制

以斯科特·费泽公司的拉尔夫为例，报酬计划5分钟敲定，无律师或薪酬顾问参与，核心概念极为简单：

> 💰 **资本不免费**

| 增量资本回报情况       | 对拉尔夫奖金的影响                |
| ---------------------- | --------------------------------- |
| 回报 **高于** 费用成本 | 奖金 **增加**                     |
| 回报 **低于** 费用成本 | 差价记在拉尔夫头上，奖金 **扣减** |
| 资金无良好去向         | 交由奥马哈总部安排                |

这体现了真正的 **“联盟”**：双向合伙人，同享涨跌，而非“正面我赢，反面你输”。

#### 5.3 总部的责任

肩负整体责任的只有巴菲特和芒格，因此只有他们应按伯克希尔整体业绩获得报酬。但他们已表示：即便没有工资，也乐于在这田园诗般的氛围中工作。



### 六、布朗鞋业：不寻常的报酬机制

1991年收购的H.H.布朗鞋业，其报酬机制极为独特：
- 高管年薪仅 **$7,800**，外加公司利润**扣除资本成本后**的分成
- 经理人与股东立场完全一致：渴望发挥才能的管理层通常是具有才能的管理层

🏭 尽管制鞋业竞争激烈、库存积压严重，布朗鞋业凭借这一机制，在利润率和资产回报率上均创下非凡历史纪录。



### 七、CEO薪酬失控与董事会责任

#### 7.1 薪酬委员会如何丧失立场

CEO与薪酬委员会的谈判，往往变成“玩钱”：
- CEO对10万还是50万期权极为在意；薪酬委员会却觉得差别不大（尤其财务报告影响不明显时）
- 贪婪方案通过顾问和人力部门迅速复制——“有两类客户你不能冒犯：现实的和潜在的。”
- 薪酬委员会像滑雪道上的小狗，乖乖跟着顾问建议，忘记忠诚于股东

> “如果你无法分辨一些人的立场，通常他们就不是站在你这一边的人。”

#### 7.2 从“更多”到“减少”

19世纪90年代工会领袖喊出“更多！”，20世纪90年代美国CEO们沿用了这一口号。结果是：CEO们积累丰厚财富，股东却经历财务灾难。

🚫 董事们必须停止这种抢劫：
- 薪酬委员会成员应**像花自己的钱一样**对待股东的钱
- 应为股东谈判，明确解释薪酬逻辑与业绩衡量方式
- 对杰出表现给丰厚奖励没错，但若无杰出表现，就该大声喊出 **“减少薪酬”**



### 总结：回归理性的薪酬之道

1. **留存利润自动增长不是管理成就**，激励机制应剔除这种“自动扶梯”效应。
2. **固定价格期权漠视资本成本**，创造的是“正面我赢，反面股东输”的不对等赌局。
3. **报酬必须与可控业绩挂钩**，分清局部责任与整体责任。
4. **资本必须计入成本**，形成双向对称的奖惩。
5. **董事会必须代表股东**，而非屈服于管理层和顾问，终结以“更多”为口号的掠夺。

伯克希尔的实践表明：真正与股东站在同一立场，只需简单、理性的原则——以及把资本当成有成本的东西。



## G.风险、声誉和失察

### 一、 CEO对风险控制承担绝对责任

> ⚠️ **核心原则：风险控制责任不可转移。**

#### 1.1 伯克希尔模式
- **最终责任人**：巴菲特本人负责起草和监控公司几乎所有的衍生品合约，仅少数与运营相关的合约除外。
- **问责机制**：若公司因风险失控而陷入困境，责任完全归于CEO，不应推诿给风险委员会或首席风险官。
- **董事会职责**：在大型金融机构中，董事会若不能确保CEO对风险控制负全责，即属**玩忽职守**。

#### 1.2 金融机构惨痛教训的启示
2008年金融危机中，许多大型金融机构的股东承受了毁灭性损失，但其CEO却未付出相应代价。这个现象揭示了责任与后果的严重不对等。

| 项目         | 详情                                                         |
| :----------- | :----------------------------------------------------------- |
| **损失规模** | 仅四家最大的金融机构，便抹去股东财富高达 **5,000亿美元**。   |
| **股东损失** | 在许多失败案例中，股东持股价值损失 **90%** 或更多。          |
| **CEO处境**  | 他们虽然个人财富受损，但依然过着优渥的生活，未承担对等的职业和财务惩罚。 |
| **建议准则** | 若机构和国家因其鲁莽行为受损，CEO和董事**本人也应付出巨大代价**，且该损失不能由原公司或保险公司进行补偿。他们应品尝“大棒”的滋味。 |



### 二、 声誉至上：名誉胜过金钱

> 💡 **最高优先原则：积极热烈地捍卫伯克希尔的名声。**

#### 2.1 不可妥协的红线
巴菲特指出：“**我们可以承受金钱的损失，甚至损失大笔金钱，但我们无法承受名誉的损失，甚至一丝一毫也不可以。**”

#### 2.2 行为评估的“头版测试”法
在评估任何商业行为时，不应仅判断其**是否合法**，更需思考：
> “如果这个行为被一个不友好却聪明的记者撰写，并刊出在国家报纸的头版上，我们是否还能愉快面对？”

#### 2.3 拒绝“大家都在做”的借口
当一项决策需要靠“每个人都这么干”来辩解时，它几乎必然存在问题。
- **道德评估**：此借口完全不可接受，它暴露了决策者无法给出正当理由的窘境。
- **终极追问**：让提出此辩解的人，把同样的话对记者或法官再说一遍，答案自明。
- **行动准则**：若对一项行为的合规性或合理性心存“迫近底线”的疑虑，**直接放弃它，就当这笔钱不属于你。**



### 三、 直面坏消息，塑造坦率文化

#### 3.1 及时预警，阻止问题蔓延
巴菲特的处理原则是：**立即通报任何重大坏消息**。
- **教训案例**：所罗门证券公司起初的一个可轻易解决的问题，因管理层不愿迅速直面坏消息，导致丑闻不断发酵，几乎令这家拥有8,000名员工的公司消亡。
- **管理信条**：管理者对坏消息的态度，是企业文化形成的最关键因素。**是文化，而不是文件，更能决定一个企业的行为。**

#### 3.2 文化的力量
在伯克希尔25万名员工中，每日无人出错的可能性为零。关键是在最细微的不良气息出现时，利用巨大的影响力将问题消灭在萌芽状态。管理者的行为和言语，就是塑造这种上行下效文化的最有效工具。



### 四、 审计委员会的四个关键质询

审计委员会的核心任务并非关注架构章程，而是要让审计师敢于披露真相。近年来，审计师惯于视CEO为客户，为了保住业务而不敢冒犯管理层。想要打破这种舒适关系，审计委员会必须明确向审计师指出，**瞒报知情的问题将面临巨额罚金**。

为此，审计委员会可通过向审计师提出以下四个关键问题，并要求将答复记录在案呈报给股东，以此起到强大的**预防作用**。对牢狱之灾的畏惧，将从源头制止不当行为。

| 序号  | 审计委员会质询问题                                           | 目的与深层含义                                               |
| :---: | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **1** | 如果审计师**全权负责**编制公司的财务报表，他们的最终成果是否会与现管理层准备的报表有所不同？ | 迫使审计师披露所有实质性或非实质性的区别，揭示管理层可能存在的会计选择偏差。 |
| **2** | 如果审计师本人就是**投资者**，他们是否已经收到了所有用于了解报告期内公司财务表现的重要信息？ | 验证信息披露的充分性，确审计师没有隐瞒应告知投资者的关键信息。 |
| **3** | 如果审计师本人就是**CEO**，公司是否严格遵循了本应遵守的内部审计流程？若没有，有何区别与原因？ | 检查公司内控流程的实际执行情况，而非仅仅停留在书面制度。     |
| **4** | 审计师是否察觉到任何**将营业收入或成本在不同报告期之间进行转移**的行为，以及其目的和影响？ | 专门防范操纵利润、跨期调节盈余的舞弊行为，确保盈利数字的真实可信。 |

这一流程将迫使审计师从一开始就拒绝为违规行为背书，从而防患于未然。
