## 一、核心理念

巴菲特认为，**买下整个企业和以购买股票的形式买一部分企业，本质上没有根本性区别**。他倾向于直接拥有整个企业，以便对企业的资本配置拥有决定权；但购买部分股票也有其优势——投资选择更多，且更容易发现价格低估的机会。

> 💡 **投资视角**：投资时应以**企业分析师**的眼光审视企业，而非市场分析师、宏观经济分析师或股票分析师的眼光。

巴菲特说："我是一个好的投资者，因为我是一个企业家；我是一个好的企业家，因为我是一个投资者。"

## 三、企业准则（Business Tenets）

### 3.1 企业是否简单易懂？

**核心观点**：投资回报与投资者对企业的理解程度成正比。

- **能力圈原则**：只投资自己能完全理解的企业，包括营收、成本、现金流、劳资关系、价格弹性及资本配置需求等
- **专注主业**：优秀企业的显著特点是专注主业，多元化和复杂往往导致恶化
- **投资智慧**："投资成功并不是靠你懂多少，而是认清自己不懂多少"

> 💡 **实践建议**：消费、医药类企业和生活息息相关，较容易理解；科技类企业技术迭代快，不易长期持有。

### 3.2 企业是否有持续稳定的经营历史？

**核心观点**：规避复杂，规避陷入麻烦的企业。

- **避开转型企业**：经历重大变化的企业会增加犯错的可能
- **优选稳定企业**：最好的回报来自那些**多年稳定经营、提供同样产品或服务**的公司
- **巴菲特箴言**："一个伟大企业的定义至少是'伟大'期间能持续**25-30年**"

### 3.3 企业是否有良好的长期前景？

**核心观点**：经济世界分为不等的两部分——少数伟大的企业（特许经营权）和平庸的企业。

**特许经营权企业的三大特征**：

| 特征             | 说明                       |
| ---------------- | -------------------------- |
| **被需要或渴望** | 产品或服务具有稳定消费需求 |
| **无可替代**     | 没有替代产品或竞争对手     |
| **没有管制**     | 可以自主定价，不受价格管制 |

**价格弹性**：这类企业可以保持售价或偶尔提高售价，不用担心失去市场份额和销量，从而获得超出平均的资本回报。

**普通商品型企业**：提供的产品实质上是普通型商品（如石油、化学品、电脑、汽车、航空服务、银行等），无法与竞争者区分，只能在价格上竞争，损害利润。

## 四、管理准则（Management Tenets）

### 4.1 管理层是否理性？

**核心能力**：配置公司资本的能力是管理层最重要的能力，长期而言将决定股东价值。

**理性资本配置的两种方式**：

1. **分红或提高分红**——当找不到超越平均回报率的投资项目时
2. **回购股票**——当股票被低估时

> ⚠️ **非理性表现**：将留存利润投资于低回报项目，仅仅为了填满时间或满足扩张欲望。

### 4.2 管理层对股东是否坦诚？

**核心品质**：承认错误、分享成功，全面真实反映公司状况。

**坦诚管理的三大价值**：

1. 帮助股东回答：**公司的大致估值是多少？**
2. 帮助股东回答：**公司多大可能性达到未来的目标？**
3. 帮助股东回答：**鉴于过去的表现，管理层干得如何？**

> 💡 **巴菲特观点**："在公众场合误导他人的CEO，在私底下也会误导自己。"

### 4.3 管理层能否抗拒惯性驱使？

**惯性驱使的四大来源**：

| 来源             | 具体表现                                       |
| ---------------- | ---------------------------------------------- |
| **组织拒绝改变** | 拒绝改变当前方向，即使需要转型                 |
| **闲不住**       | 仅仅为了填满时间，通过新项目或并购消化现金     |
| **满足领导愿望** | 无论项目多么愚蠢，下属都会准备好可行性报告     |
| **盲目模仿同行** | 同行的扩张、收购、薪酬计划都会引起没头脑的模仿 |

**根本原因**：人性——大多数管理层不想被看起来愚蠢，不断与同业比较，高估自己的能力。

> 💡 **伯克希尔原则**：正确与否**不仅仅因为大家都这么做**。

## 五、财务准则（Financial Tenets）

### 5.1 重视净资产回报率（ROE），而不是每股盈利（EPS）

**核心逻辑**：

- 每股盈利保留了以往留存下来的利润，随着企业运营不断增加
- **ROE（盈利÷股东权益）**能更真实反映企业的盈利能力，因为它考虑了逐年增加的资本额
- **黄金标准**：不少价值投资者首先排除长期ROE < 15%的企业

**计算调整**：

1. 所持有的证券以**成本计算**而非市价计算
2. 剔除**非经常性项目**（资本损益、特别年度表现）
3. 关注**无负债或极少负债**情况下产生的回报

> ⚠️ **警惕**：有些企业通过提高资产负债率来提高ROE，巴菲特不喜欢这种做法。

### 5.2 计算真正的"股东盈余"

**公式**：

```
股东盈余 = 净利润 + 折旧 + 损耗 + 摊销 - 资本支出 - 其他必需的营运资本
```

**核心观点**：

- **利润表利润常常是"虚的"**——重资产型企业会被通货膨胀悄悄侵蚀
- **现金流也不完美**——对于持续需要资本支出的制造业，现金流会误导投资者
- **轻资产行业更受青睐**——重资产行业会不断吞噬企业利润，都是"纸面财富"

> 💡 **巴菲特引用凯恩斯**："宁要模糊的正确，不要精确的错误。"

### 5.3 寻找具有高利润率的企业

**核心逻辑**：

- 高利润率反映企业具有**经济特许权**
- 提升盈利能力没有大秘密：**控制成本**
- 优秀经理人把控制成本像呼吸一样自然必需

**巴菲特原话**：

> "真正优秀的经理人不会在早上醒来说：我今天要削减成本。就像他醒来后不会专门想着需要呼吸一样。"

### 5.4 每一美元的留存利润，至少创造一美元的市值

**"一美元原则"**：

- 公司每留存一美元的利润，至少应该创造**一美元的市场价值**
- 如果市值提升超过留存数字，则更好
- 长期而言，股票市场反映企业的合理价值

**检验方法**：将净收入减去股利得出保留盈余，对比公司现在与10年前（或4-5年平均值）的市价差价。

## 六、市场准则（Market Tenets）

### 6.1 必须确定企业的市场价值

**估值方法——现金流折现法（DCF）**：

> "一个公司的价值决定于在其生存期间，预期产生的所有现金流，在一个合理的利率上的折现。"  
> ——约翰·伯尔·威廉姆斯《投资价值理论》

**两个关键变量**：

| 变量       | 说明                                                         |
| ---------- | ------------------------------------------------------------ |
| **现金流** | 企业未来股东盈余现金流                                       |
| **贴现率** | 巴菲特使用**美国政府长期国债利率**（无风险收益率），不加风险溢价 |

**核心原则**：

- 只选择**简单稳定**的企业，保证数据高度确定性
- 用最保守的方法估计企业价值，以降低风险
- 投资者应该买入以现金流贴现方法计算后，**最便宜的股票**（越便宜越好）

### 6.2 相对于企业的市场价值，能否以折扣价格买到？

**安全边际理论**：

- **定义**：在购进价格和企业价值之间要有足够的"安全边际"
- **作用**：缓冲效果可保证不受公司未来现金流量变动影响
- **买入时机**：大企业暂时出现危机或股市下跌时是大量买进的时机

**安全边际的双重保护**：

1. **为股价下跌提供保护**——即使企业价值意外地减少，原始买入价仍能产生足够回报
2. **确保本金安全**——在确保本金安全的情况下，取得令人满意的回报

> ⚠️ **巴菲特警告**：即便是伟大的企业，以过高的价格买入也会导致收益的平庸。

## 七、投资错误的可能来源

巴菲特指出，如果投资犯错，可能源于以下三方面：

| 错误来源               | 说明                               |
| ---------------------- | ---------------------------------- |
| **支付的价格**         | 买入价格过高，安全边际不足         |
| **管理层**             | 管理层不够理性、坦诚或独立思考     |
| **企业未来的经营状况** | 对企业未来现金流或竞争优势判断失误 |

> ⚠️ **最常见错误**：第三类错误——对企业未来经营状况的估算失误。

## 八、长期股价剖析

**核心公式**：

```
长期股价表现 = 优秀的企业 + 合理的价格 + 良好的管理
```

**关键逻辑**：

- 一个伟大的企业，如果买在 favorable price（有利价格）并由 sound management（稳健管理）经营，将产生**持续上升的股东价值**
- 短期而言，股价可能因某些原因上下波动背离其价值
- 长期而言，股票市场会**纠正错误，回归正常**，反映企业的合理价值
