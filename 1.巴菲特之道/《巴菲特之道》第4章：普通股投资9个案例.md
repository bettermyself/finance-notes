## 一、本章核心框架

💡 **巴菲特的普通股投资逻辑**：以**企业所有者**的视角，寻找具备**持久竞争优势（护城河）**、**优秀管理层**和**合理估值**的公司，长期持有并分享企业成长。

| 核心维度       | 关键标准                                     |
| -------------- | -------------------------------------------- |
| **商业模式**   | 简单易懂，能在5分钟内解释清楚盈利方式        |
| **经济护城河** | 品牌、成本优势、网络效应、牌照壁垒、转换成本 |
| **财务指标**   | 高ROE（≥15%）、低负债、充裕现金流            |
| **管理层品质** | 理性、诚信、资本配置能力卓越                 |
| **估值原则**   | 留有安全边际，价格低于内在价值               |

## 二、九大投资案例详解

### 2.1 华盛顿邮报公司（The Washington Post Company）

| 项目             | 详情                                            |
| ---------------- | ----------------------------------------------- |
| **首次投资时间** | 1973年                                          |
| **买入背景**     | 水门事件引发报业危机，市场恐慌性抛售            |
| **核心逻辑**     | 拥有垄断性地方报纸 + 电视台广播牌照，具备定价权 |
| **关键数据**     | 市值约8,000万美元，巴菲特估算内在价值4-5亿美元  |
| **投资成果**     | 持有数十年，收益率超百倍                        |

**投资要点**：
- ⚠️ **市场情绪极端悲观时，往往是最佳买入时机**——华盛顿邮报的困境是行业性的，而非公司自身护城河被破坏
- 重视**无形资产价值**：品牌声誉与特许经营权带来的定价能力
- 凯瑟琳·格雷厄姆展现了卓越的资本配置能力和对股东的坦诚态度

## 四、现金流贴现与股东盈余 ÷ 无风险利率

### 4.1 核心框架：两种方法的内在统一

💡 **巴菲特估值的本质**：无论是复杂的DCF模型，还是简化的"股东盈余 ÷ 无风险利率"，核心都是**将企业未来产生的现金回流，折现到当前时点**。

| 方法                      | 适用场景                   | 复杂度 | 巴菲特的使用频率       |
| ------------------------- | -------------------------- | ------ | ---------------------- |
| **现金流贴现（DCF）**     | 现金流稳定、可预测的企业   | 中等   | 概念性理解，非机械计算 |
| **股东盈余 ÷ 无风险利率** | 永续经营、护城河深厚的企业 | 极简   | 日常心智计算的主要工具 |

> ⚠️ **关键认知**：两种方法在数学上是等价的。当假设"永续零增长"时，DCF公式就简化为"股东盈余 ÷ 无风险利率"。

---

### 4.2 现金流贴现模型（DCF）

#### 4.2.1 标准DCF公式

$$
\text{企业内在价值} = \sum_{t=1}^{n} \frac{FCF_t}{(1+r)^t} + \frac{FCF_{n+1}}{(r-g)} \times \frac{1}{(1+r)^n}
$$

| 符号    | 含义              | 巴菲特的实践调整                             |
| ------- | ----------------- | -------------------------------------------- |
| $FCF_t$ | 第t年的自由现金流 | 使用**股东盈余**替代标准FCF                  |
| $r$     | 折现率            | **无风险利率**（长期国债利率），不加风险溢价 |
| $g$     | 永续增长率        | 保守假设，通常≤3%或等于通胀率                |
| $n$     | 详细预测期        | 5-10年，之后进入永续增长                     |

#### 4.2.2 巴菲特的DCF简化实践

巴菲特**从不做复杂的Excel建模**，他的DCF是**心智层面的模糊计算**：

> "如果一项投资需要精确到小数点后两位的计算才能决定，那我就不应该做这笔投资。"

**他的简化步骤**：
1. **估算未来5-10年的平均股东盈余**
2. **假设一个保守的永续增长率**（通常为零或极低）
3. **使用无风险利率作为折现率**
4. **与当前市值比较**，留出安全边际

---

### 4.3 股东盈余 ÷ 无风险利率

#### 4.3.1 公式推导：从DCF到简化公式

当假设企业**永续零增长**（$g = 0$）时，DCF公式简化为：

$$
\text{企业内在价值} = \frac{\text{股东盈余}}{r}
$$

这就是巴菲特最常用的**"简易估值法"**。

| 参数                 | 定义               | 获取方式                         |
| -------------------- | ------------------ | -------------------------------- |
| **分子：股东盈余**   | 企业真实可分配现金 | 净利润 + 折旧摊销 - 必要资本支出 |
| **分母：无风险利率** | 资本的机会成本     | 长期美国国债到期收益率           |

#### 4.3.2 公式背后的经济直觉

将企业视为一张**"永续债券"**：
- **票面利息** = 股东盈余（每年产生的真实现金）
- **市场利率** = 无风险利率（替代投资的机会成本）
- **债券价值** = 利息 ÷ 利率

💡 **示例**：如果一家企业每年产生10亿美元股东盈余，长期国债利率为5%，则：
$$
\text{内在价值} = \frac{10亿}{5\%} = 200亿美元
$$

---

### 4.4 两种方法的对比与选择

| 对比维度         | 现金流贴现（DCF）     | 股东盈余 ÷ 无风险利率            |
| ---------------- | --------------------- | -------------------------------- |
| **增长假设**     | 可包含增长阶段        | 假设零增长或极低增长             |
| **适用企业**     | 增长可预测的企业      | 护城河深厚、现金流稳定的成熟企业 |
| **计算复杂度**   | 需要预测多期现金流    | 单期计算，心智可完成             |
| **对利率敏感**   | 是（折现率影响终值）  | 是（分母直接决定估值）           |
| **巴菲特偏好**   | 概念理解              | **日常决策主力工具**             |
| **安全边际来源** | 保守假设增长率/折现率 | 要求市场价格显著低于计算结果     |
